# RFC1918 Access Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Remove RFC1918 private IPv4 ranges from nanobot's blocked-network guard so WebUI tool calls can access intranet targets by default.

**Architecture:** Add a WebUI startup patch that mutates `nanobot.security.network._BLOCKED_NETWORKS` after imports are loaded. Keep the change local to this repository instead of editing the installed package directly. Verify the shared network guard now allows RFC1918 while still blocking loopback and link-local targets.

**Tech Stack:** Python 3.11+, pytest, WebUI startup monkey patches, `nanobot-ai`

---
