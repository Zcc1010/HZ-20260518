## 关键执行细则

1. **整章删除**：技术参数（第2章）、硬件原理/硬件描述（第4章）、人机接口（第6章）、通信（第7章）、安装调试（第8章）、运行监视（第9章）— 标题到下一个 `#` 大章节之间全部删除
2. **按编号定位**：在 MinerU md 中搜 `^## 第 N 章`（含中英文空格差异），识别出三大章节（1/2/3 章号）
3. **保留范围**：
   - 概述 = 第1章内，**去除** 1.3 性能特征、1.4 执行标准、1.5 订货须知
   - 保护原理 = 第3章全章
   - 定值说明 = 第5章内，**去除** 测控参数节、装置设置节
4. **找不到完整三章时**：以搜到的章节为准，找不到就空着不输出

## 排除项分组速查

**概述内排除**：
- ~~性能特征~~（装置硬件性能描述，与定值校核无关）
- ~~产品执行标准~~

**保护原理内排除**：
- ~~测控功能~~（遥测/遥信/遥控，非保护逻辑）
- ~~时间管理/时钟同步~~（对时方式，非保护逻辑）

**定值说明内排除**：
- ~~测控参数~~（遥信参数、遥测参数、遥控参数、同期参数）
- ~~装置设置~~（装置参数、辅助参数、对时方式、通信参数、模拟量显示）

**整章不提取**：
- ~~技术参数~~、~~硬件描述~~、~~人机接口~~、~~运行说明~~、~~调试大纲~~
