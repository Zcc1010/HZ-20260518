# 格式规范

> PDF 提取合并阶段，需将双源 md 重写为统一格式。本文件汇总所有格式规则。
> **`scripts/extract-manual.ts` 已实现本文件的所有自动转换**，调用脚本即可，不必手写。

## 1. 标题层级（自动修正）

MinerU 把所有章节都标成 `##`，不区分层次。脚本按章节号位数自动重映射：

| 编号位数 | 应有层级 | Markdown | 示例 |
| --- | --- | --- | --- |
| `N`（章） | h2 | `## N` | `## 1 概述`、`## 3 工作原理`、`## 5 定值及参数` |
| `N.M`（节） | h3 | `### N.M` | `### 1.1 适用范围`、`### 3.1 保护功能` |
| `N.M.K`（小节） | h4 | `#### N.M.K` | `#### 3.1.1 启动元件`、`#### 5.1.2 保护定值` |
| `N.M.K.L`（小小节） | h5 | `##### N.M.K.L` | |

**特殊处理**：
- 标题最外层用 `#`（说明书标题）
- 源里漏了 `##` 的纯文本章节标签（如 `5.1.1 设备参数`）也按编号位数补上
- 误标成 `##` 的中文列表项（如 `## 1）重合闸...`）自动去掉 `##` 还原为列表项

## 2. 表格：HTML → Markdown（自动转换）

- HTML `<table>` 全部转 Markdown 表格 `| 列1 | 列2 |`
- **rowspan 展开**：合并单元格的值**重复填到每一行**（如"保护功能"重复 12 次）
- **colspan 展开**：合并单元格的值**横向重复 N 列**（如"类别"列注脚 colspan=6 展开为 6 个相同内容）

> **已知数据瑕疵**（源 PDF 提取导致，markdown 格式已正确）：
> - 5.1.2 表"1617"序号合并——源 HTML 损坏，`<td>16</td><td>17</td>` 在同一 `<tr>`
> - 5.4 表"出口名称保护动作"列名合并——源 PDF 列名提取错
> - colspan=6 注释行重复 6 次——是 colspan 展开的副作用

## 3. 列表项清理

PDF 提取的特殊字符列表标记统一转为 `- `：

- Wingdings 符号（U+F06C、U+F0B7 等）→ `- `
- 行首 `. ` → `- `
- 行尾双空格连续列表项 → 独立列表行
- `## 1）...` 等误标 `##` 的中文列表项 → 还原为 `1）...`

## 4. 图片处理（严格）

- **只保留被 md 引用的图片**：脚本扫描最终 md 里的 `![](images/HASH.jpg)`，只复制这些
- **不依赖图号**判断（OCR 容易把"图 3-1-1"识别错），只跟实际引用走
- 复制图片从 `temp/{型号}/mineru/images/` 复制到 `{输出目录}/images/`
- md 中用相对路径 `![](images/xxx.jpg)` 引用
- **节省效果**：未引用的图片（封面、目录、扫描残留、其它章节）全部不复制

## 5. `<details>` 块清理（自动）

MinerU 输出的两种 details 块**全部删除**：

| summary 类型 | 内容 | 处理 |
| --- | --- | --- |
| `flowchart` | ` ```mermaid...``` ` | 删除整个 `<details>` 块，**保留块前的图片引用** |
| `text_image` | OCR 文字备份（如相量图 OCR 不出图的回退） | 删除整个 `<details>` 块 |

> **正则陷阱**：必须严格匹配 `summary=flowchart` 或 `summary=text_image`，不能用 `[\s\S]*?` 非贪婪跨块匹配——会误把 text_image 和 flowchart 一起吞了。

## 6. 其他清理

- **删除** `<details>` 块、mermaid 代码块
- **删除** 页眉页脚、目录页、重复章节标题
- **保留** LaTeX 公式（`$$...$$` 和 `$...$`）

## 7. 原版 PDF 复制（可选）

建议在 `extract-manual.ts` 调用时传入原版 PDF 路径，脚本会复制到输出目录：

```bash
npx tsx scripts/extract-manual.ts PAC-8211A "/path/to/PAC-8211A-GZK(提升)线路保护测控装置技术说明书.pdf"
```

**目的**：图片是 MinerU 从 PDF 抽出来的，md 表格是 OCR 文字。原版 PDF 是"图和文字是否对得上"的唯一溯源依据，校核人员遇到疑问时可以查 PDF 原文。
