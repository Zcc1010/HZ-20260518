## 一次值换算规则

一次值(kA) = 二次值(A) × (CT一次额定值 / CT二次额定值) / 1000

CT参数来源：HDR SettingValue模块。

---

## 时间对齐计算方法

### 两步对齐法

**第一步：电流正突变对齐**
1. 找出每个装置故障相电流正突变值最大的一相及其绝对时间
2. 选择所有装置中正突变时间最早的为参考时间
3. 各装置校正量 = 参考时间 - 该装置正突变时间
4. 无电流正突变的装置，用同厂站第一次负突变对齐

**第二步：保护启动对齐**
1. 将校正量应用到各装置的保护启动时间，得到校正后的保护启动时间
2. 选择校正后最早的保护启动时间为t=0ms
3. 所有事件的相对时间 = 校正后事件时间 - t=0ms基准时间

### 示例

四个装置电流正突变绝对时间：T1, T2, T3, T4（T1最早）
参考时间 = T1
校正量：C1=0, C2=T1-T2, C3=T1-T3, C4=T1-T4

四个装置保护启动绝对时间：S1, S2, S3, S4
校正后保护启动：S1+C1, S2+C2, S3+C3, S4+C4
最早校正后保护启动 = t=0ms基准
