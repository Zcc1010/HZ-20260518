# 装置本体缺陷 (device_body)

## 类别说明

装置本体缺陷指保护装置自身硬件或软件异常导致的告警或闭锁状态。

## 相关信号类型

| 标准标签 | 说明 |
|----------|------|
| DEVICE_ALARM | 装置异常告警 |
| DEVICE_SELF_CHECK | 装置自检 |
| DEVICE_BLOCKED | 装置闭锁 |
| HARDWARE_FAULT | 硬件故障 |
| SOFTWARE_FAULT | 软件异常 |

## 典型规则

### Rule 01001: 装置硬件自检异常

**判定条件**：
1. 装置发出硬件自检异常告警
2. 包含硬件类型标识（CPU、内存、Flash等）
3. 持续时间超过设定阈值

**严重程度**：危急

**整改建议**：
- 记录告警详细信息
- 联系厂家进行硬件更换
- 备用装置投入运行

---

### Rule 01002: 装置程序版本异常

**判定条件**：
1. 检测到程序版本校验失败
2. 或程序版本与备案版本不一致

**严重程度**：一般

**整改建议**：
- 核对程序版本
- 重新下载正确版本
- 履行版本变更手续

---

### Rule 01003: 装置闭锁状态

**判定条件**：
1. 装置处于闭锁状态
2. 保护功能全部或部分被闭锁
3. 非计划性闭锁（非人工操作）

**严重程度**：危急

**整改建议**：
- 查明闭锁原因
- 消除闭锁条件
- 恢复保护功能
