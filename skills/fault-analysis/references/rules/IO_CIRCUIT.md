# 开入开出回路异常 (io_circuit)

## 类别说明

开入开出回路异常指保护装置的开入量（DI）和开出量（DO）回路异常。

## 相关信号类型

| 标准标签 | 说明 |
|----------|------|
| TRIP_OUTPUT | 跳闸出口 |
| CLOSE_OUTPUT | 合闸出口 |
| INPUT_SIGNAL | 开入量 |
| OUTPUT_SIGNAL | 开出量 |
| TWJ_POSITION | 跳闸位置 |
| HWJ_POSITION | 合闸位置 |

## 典型规则

### Rule 05001: 跳闸出口未动作

**判定条件**：
1. 保护装置发出跳闸命令
2. 跳闸出口信号未出现
3. 断路器未跳闸

**严重程度**：危急

**整改建议**：
- 检查跳闸出口压板
- 检查跳闸回路
- 检查操作箱/操作机构

---

### Rule 05002: 跳闸出口异常动作

**判定条件**：
1. 无保护跳闸命令
2. 但跳闸出口动作
3. 或跳闸出口持续动作

**严重程度**：危急

**整改建议**：
- 检查保护装置逻辑
- 检查跳闸回路绝缘
- 检查出口继电器

---

### Rule 05003: 开入量异常

**判定条件**：
1. 开入量状态与实际不符
2. 或开入量抖动
3. 或开入量丢失

**严重程度**：严重

**整改建议**：
- 检查开入量回路
- 检查辅助触点
- 检查装置光耦输入
