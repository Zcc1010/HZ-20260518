# 一二次设备状态不对应 (primary_secondary)

## 类别说明

一二次设备状态不对应指一次设备（如刀闸、断路器）的位置状态与二次系统的指示或电流电压状态不一致。

## 相关信号类型

| 标准标签 | 说明 |
|----------|------|
| DISCONNECTOR_POS | 刀闸位置 |
| BREAKER_POS | 断路器位置 |
| CURRENT_FLOW | 电流流动 |
| VOLTAGE_PRESENT | 电压存在 |

## 典型规则

### Rule 02001: 刀闸位置与电流不符

**判定条件**：
1. 刀闸分位
2. 但检测到电流
3. 超过设定阈值

**严重程度**：严重

**整改建议**：
- 检查刀闸实际位置
- 检查刀闸辅助触点
- 检查电流回路

---

### Rule 02002: 断路器位置与电流不符

**判定条件**：
1. 断路器分位（TWJ动作）
2. 但检测到负荷电流
3. 超过设定阈值

**严重程度**：危急

**整改建议**：
- 检查断路器实际位置
- 检查断路器辅助触点
- 检查保护跳闸出口
