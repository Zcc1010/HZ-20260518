# 保护动作行为异常 (protection_behavior)

## 类别说明

保护动作行为异常指保护装置的动作逻辑、时序或结果不符合预期。

## 相关信号类型

| 标准标签 | 说明 |
|----------|------|
| DIFFERENTIAL_TRIP | 差动保护跳闸 |
| DISTANCE_TRIP | 距离保护跳闸 |
| ZERO_SEQUENCE_TRIP | 零序保护跳闸 |
| PROTECTION_START | 保护启动 |
| TRIP_OUTPUT | 跳闸出口 |
| RECLOSE | 重合闸 |

## 典型规则

### Rule 03001: 差动保护误动

**判定条件**：
1. 差动保护动作
2. 区外故障或正常负荷状态
3. 差流未超过整定值

**严重程度**：危急

**整改建议**：
- 检查CT极性
- 核对整定值
- 检查二次回路

---

### Rule 03002: 差动保护拒动

**判定条件**：
1. 区内故障
2. 差流超过整定值
3. 差动保护未动作

**严重程度**：危急

**整改建议**：
- 检查保护装置功能
- 检查跳闸回路
- 核对整定值

---

### Rule 03003: 距离保护异常动作

**判定条件**：
1. 距离保护动作阻抗异常
2. 与故障位置不符
3. 或超越/欠范围

**严重程度**：严重

**整改建议**：
- 检查阻抗测量
- 核对整定值
- 检查PT/CT回路

---

### Rule 03005: 重合闸异常

**判定条件**：
1. 重合闸条件满足但未重合
2. 或重合闸不应重合但重合
3. 重合闸时间异常

**严重程度**：一般

**整改建议**：
- 检查重合闸逻辑
- 核对重合闸条件
- 检查闭锁信号

---

### Rule 03006: 两套保护动作一致性异常

**判定条件**：
1. 同侧两套保护动作时间差 > 20ms
2. 或返回时间差 > 20ms
3. 或持续时间差 > 10ms

**严重程度**：严重

| 接地距离Ⅰ段(第一次) | — | — | — | — | — | — | >20ms | >20ms | >10ms | [待核实] |
| 重合闸动作 | — | — | — | — | — | — | | | |
| 纵联差动保护(第二次) | — | — | — | — | — | — | | | |
| 距离加速 | — | — | — | — | — | — | | | |

**一致性判定规则**：
- 动作时间差 > 20ms → [待核实：动作时间不一致]
- 返回时间差 > 20ms → [待核实：返回时间不一致]
- 持续时间差 > 10ms → [待核实：持续时间差XXms>10ms]

**整改建议**：
- 检查保护装置采样和处理机制差异

- 检查软压板投入情况
- 检查CT变比和接入位置是否一致

### Rule 03007: 两套保护启动时间不一致

**判定条件**：
同侧两套保护启动时间差 > 20ms，或一套启动另一套未启动

**严重程度**：严重

**分析要求**：
1. **提示不一致**：在一致性校验表中标注启动时间差异
2. **分析启动机制**：根据装置型号和说明书，分析启动快慢的机制差异
3. **重点分析闭锁条件**：检查是否有满足的闭锁条件导致启动延迟或启动失败

**重点：零序电流异常导致启动元件闭锁**

当自产零序电流与外接零序电流差异较大时，可能触发自检异常，导致启动元件被闭锁。这是启动慢或未启动的**主要原因**。

**结合装置说明书分析启动慢/未启动的原因**：

| 优先级 | 可能原因 | 说明 |
|-------|---------|------|
| 1 | **零序电流异常** | 自产零序电流与外接零序电流差异触发自检异常，闭锁启动元件 |
| 2 | 启动定值（IQD）配置过高 | 灵敏度不足，低电流故障时无法启动 |
| 3 | 通道同步机制 | CSC-103A-G需先建立通道同步，导致启动延迟 |
| 4 | PT断线检测影响 | PT断线检测在启动元件不启动时投入 |
| 5 | 远方跳闸受启动元件控制 | "远跳受启动元件控制"=1时，需本装置启动才出口 |
| 6 | 弱电源侧特性 | 相间电压条件未满足，辅助启动未动作 |

**启动元件未动作的可能原因（结合说明书）**：
1. **自产零序电流与外接零序电流差异**：触发自检异常，闭锁启动元件（主要原因）
2. **启动定值（IQD）配置过高**：灵敏度不足
3. **冲击性负荷控制字投入**：抑制突变量启动
4. **PT断线检测影响**：启动后PT断线检测停止，但启动前可能影响
5. **远方跳闸受启动元件控制**："远跳受启动元件控制"=1时，需本装置启动才出口
6. **弱电源侧特性**：收到对侧允许信号但相间电压条件未满足

**一致性判定规则**：
- 启动时间差 > 20ms → `[待核实: 启动时间不一致]`
- 动作元件差异（如一套有差动另一套无）→ `[待核实: 启动元件未动-零序电流异常可能]`

**整改建议**：
- **首要检查**：自产零序电流与外接零序电流的一致性（最可能原因）
- 检查变化量启动电流定值（IQD）配置
- 检查"远跳受启动元件控制"定值
- 检查通道配置和通信状态

### Rule 03008: 主变后备越级

**触发条件**:

- **前置 P1**:主变后备保护任一动作(高压侧复压方向过流 / 高压侧零序过流 / 中压侧复压过流 / 中压侧零序过流 / 低压侧限时过流)
- **前置 P2**:故障点不在主变保护区内(穿越性判据,见下)
- **主体 S1**:下级保护未动作 → 场景A
- **主体 S2**:下级保护动作但断路器拒动(BRK_POS=合)→ 场景B
- **主体 S3**:≥2 条出线不同相单相接地,线路保护未动 → 场景C
- **主体 S4**:下级保护正确动作 + 断路器跳开 + **主变后备与下级同时动作(<100ms)** → 场景D
  - 典型案例:10kV 出线跳闸同时主变低后备也跳(下级虽正确跳但主变不应同时动作)

**穿越性判据**(用于 P2):

```
条件1: |I_H + I_M + I_L| < 0.1 × |I_H|  (矢量和近似为零)
条件2: 至少 1 条出线故障电流 > 2In
条件3: 主变差流 < 0.5 × 差动动作电流定值
```

三条同时满足,判定故障不在主变区内。

**严重程度**: 严重

**整改建议**:

- 场景A/B:检查下级保护装置功能、定值、回路;检查下级断路器操作机构、二次回路;核对后备保护时限配合
- 场景C:以上 + 提交异相两点接地专项分析
- **场景D**:检查主变后备保护时限配合(下级跳闸后主变后备不应再动作);核对后备保护定值整定;检查主变后备保护投入/压板状态;**提交越级跳闸专项分析报告**

**场景 D 时间窗口**:

主变后备首次动作与下级保护首次动作的时间差 < 100ms 视为同时动作。
- 典型继电器动作时间 30-50ms + 断路器跳闸时间 30-60ms = 总计 < 100ms
- 时间偏差 > 100ms 可能为主变后备"延后误动",需单独评估

**输出字段**:

```json
{
  "rule_id": "03008",
  "triggered": true,
  "scenario": "A|B|C|D",
  "severity": "严重",
  "evidence": {
    "main_trip_outputs": [...],
    "downstream_trip_outputs": [...],
    "downstream_brk_pos": [...],
    "through_current_check": {...},
    "same_time_action": true,
    "same_time_gap_ms": 52.0
  }
}
```

**案例:35kV 板桥变 2026-02-13 04:16 越级跳闸**

- 10kV 板南线 04 开关 过流II段动作 + 重合成功(下级正确动作)
- #1主变 101 低后备过流保护同时动作(越级,场景D)
- 04:21 试送 10kV I 段 → #2主变 102 低后备又动作(越级,场景D)
- 文件名时间校正后,#1主变与 10kV 04 偏差仅 52ms(< 100ms = 同时动作)
- Rule 03008 触发,场景D,严重

### Rule 03009: CT 饱和致主变差动误动

**触发条件**:

- **前置 P1**:主变比率差动动作
- **前置 P2**:主变各侧差流 > 差动动作电流定值
- **主体 M1**:主变某侧 CT 二次电流有饱和特征:
  - 2次谐波/基波 > 15% AND 基波 > 2In
  - OR(HDR 中 CT 饱和标志 = 1)
- **主体 M2**:故障录波器显示故障点在该侧母线之外
- **主体 M3**:故障初期该侧电流有典型饱和波形

**CT 饱和检测**:复用 `latent-faults/scripts/harmonic_analysis.py`,检测窗口 0~40ms。

**严重程度**: 危急

**整改建议**:

- 检查该侧 CT 变比、极性、二次负担
- 校验差动保护抗饱和措施(比率制动、二次谐波制动)
- 必要时调整 CT 变比或更换 CT

**误报避免**:同时满足区内故障特征(阻抗 < 主变阻抗 + 差流 > 2×定值)时不判 CT 饱和误动。

### Rule 03010: 主变保护未正确动作(主变自身分析规则)

> **注**:本规则不属于"越级规则",而是"主变保护动作评价"规则,用于越级规则覆盖不到的场景(如只有主变录波,或故障点在主变区内)。

**触发条件**(任一):

- **D1**:主变差动/重瓦斯拒动(区内故障但主变未动)
- **D2**:主变保护误动(区外故障但主变动了)
- **D3**:主变各侧后备配合不当(同侧两套保护动作时间差 > 50ms)
- **D4**:主变非电量保护误动(重瓦斯/压力释放跳闸但无内部故障)

**严重程度**: 危急

**整改建议**:

- 核对保护定值与说明书
- 检查保护装置功能性
- 检查二次回路绝缘与接线
- 必要时启动保护更换评估

**使用条件**:仅在 Rule 03008/03009 不适用时触发,优先级低于越级规则。
