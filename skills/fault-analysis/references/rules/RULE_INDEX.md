## 示例规则（基于2024-06-28案例）

### rule_06_001: GOOSE通信链路异常

```markdown
# Rule 06001: GOOSE通信链路异常

## 规则来源
- 文档: 继电保护及二次回路隐患缺陷在线辨识技术规范
- 条款: C.6.2

## 判定条件
1. 检测到GOOSE总告警信号
2. 同时检测到GOOSE接收链路异常信号
3. 时间差在±100ms内

## 严重程度
严重

## 整改建议
检查GOOSE网络配置，确认光纤连接正常，检查交换机端口状态

## 证据示例
- 装置: 23号
- 时间: +95ms
- 信号: ["GOOSE总告警", "链路1GOOSE接收A网链路异常"]
```
