# 定值异常 (setting)

## 类别说明

定值异常指保护装置的整定值设置不正确或与要求不符。

## 相关信号类型

| 标准标签 | 说明 |
|----------|------|
| CURRENT_SETTING | 电流定值 |
| TIME_SETTING | 时间定值 |
| IMPEDANCE_SETTING | 阻抗定值 |
| FREQUENCY_SETTING | 频率定值 |

## 典型规则

### Rule 07001: 电流定值整定错误

**判定条件**：
1. 定值与计算书不符
2. 或与整定通知单不符
3. 或明显不合理

**严重程度**：严重

**整改建议**：
- 核对整定计算书
- 核对整定通知单
- 重新整定

---

### Rule 07002: 时间定值整定错误

**判定条件**：
1. 保护时间配合不当
2. 或时间定值与要求不符
3. 导致保护误动/拒动

**严重程度**：严重

**整改建议**：
- 检查保护配合
- 核对时间定值
- 重新整定

---

### Rule 07003: 阻抗定值整定错误

**判定条件**：
1. 阻抗定值与线路参数不符
2. 或灵敏度不满足要求
3. 导致保护范围异常

**严重程度**：严重

**整改建议**：
- 核对线路参数
- 重新计算阻抗定值
- 验证保护范围
