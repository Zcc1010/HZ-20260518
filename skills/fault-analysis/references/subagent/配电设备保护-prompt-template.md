# 配电设备保护 Subagent Prompt 模板（单套 · 10/35kV）

## 任务

给定本**配电设备保护装置**（单套，1 个装置）的录波文件（hdr + rms.csv + events.csv），按 HDR/Events/RMS 三个区块提取关键信息，生成标准化的 Markdown 段落。

> **设备范围**：10/35kV 配电设备——**电容器、电抗器、站用变**（含接地变等）。
> **单套核心**：单装置；**无重合闸**（简报不含重合闸章节）。
> **接地方式**：10/35kV 多为**小电流接地**（消弧线圈/电阻/不接地），零序故障多为选线/报警而非立即跳闸。

## 输入文件

- `*.hdr` — HDR 故障报告（XML，可能 GB18030 或 UTF-8）
- `*.rms.csv` — RMS 统计值
- `*.events.csv` — 事件列表
- `*.csv` — **原始采样 CSV**（DAT→CSV 输出，含 `时间ms` 列与三相 V/I 及专属量通道），用于「故障发展过程数据」区块
- `*.development.json` — **故障发展过程量化数据**（calculate_fault_development.py 产出），可直接引用填入区块

## 字段提取规则

### 时间格式
- 绝对时间 `HH:MM:SS.mmm`；**严禁相对时间**。
- HDR 偏移量须转绝对：`绝对时间 = 故障起始时间 + ms偏移`。

### 设备类型识别
- 关键词：电容器/电容/电抗器/电抗/站用变/接地变/所用变。
- 从 HDR 设备名称与定值结构判定具体类型。

### 装置型号 / 智能站
- 型号含 FA/DA/DG → 智能站；否则常规站。
- 制造厂家**必须**从 HDR DeviceInfo 取；无则标 `[待核实]`。

### 定值 / CT·PT 变比
- 仅从 HDR `<SettingValue>` 取；无标 `[无数据]`。
- 电容器重点：不平衡（桥差/开口三角）、差压/差流、过/欠压。
- 站用变重点：差动（若配）+ 各侧后备过流/零序。

### 一次值换算
- 电流一次值(A) = 二次值(A) × CT一次/CT二次
- 无变比时只显示二次值，标 `[无CT变比]`。

### 接地方式分支（小电流接地）
- 故障相电压↓但电流很小、三相基本对称、仅零序电压出现 → 小电流接地。
- 故障判定须区分"永久性(延时跳闸)"与"瞬时性(选线不跳)"，不可套用大电流接地"零序段立即跳闸"逻辑。

## 编码处理
HDR 可能 GB18030/UTF-8，乱码时尝试另一种编码。

## 错误处理
- 文件缺失：`[本装置文件缺失]`
- 字段无数据：`[无数据]`；可推测：`[推测为XXX]`
- HDR 故障量=0 但录波正常 → 注明"保护计算异常"，同时列录波采样值与 HDR 计算值。

## 数据来源标注
段落末尾：`> 数据来源: {hdr} + {rms} + {events}`
