# 开关保护（断路器保护）跳闸简报模板

> **适用范围**：500kV+（含 750kV、1000kV 特高压，3/2 断路器接线为主）/ 220kV 3/2 接线站的**断路器保护（开关保护）**录波分析。每个断路器配置独立断路器保护，录波常为独立文件（如 `断路器保护` / `开关保护`）。
> **与线路/主变保护解耦**：开关保护**不判故障类型**，只负责"执行跳闸 + 失灵/死区/三相不一致/充电 + 重合闸（在开关保护）"的断路器侧行为；故障类型/差流/距离判据在线路或主变保护章分析。
> **命名原则**：优先使用 IED 代码；站名/断路器编号由用户提供或推断并标 `[待核实]`。
> **判相别优先用电压**：同线路/主变保护一致，以电压特征为主。

## 故障前置条件识别（必做，写在简报开头或 §一）

解析前先判定并写入：
1. **电压等级**：PT 变比 / 断路器额定电压 / HDR 系统参数 → **500（含 750/1000kV）/ 220** kV（开关保护多见于 3/2 高电压等级）。
2. **接线方式**：**3/2 断路器**（一个半断路器串，每台断路器被两条线路共享）→ 启用本节；常规双母/单母接线一般无独立开关保护分析（N/A）。
3. **设备类型**：**开关保护（断路器保护）**，并标注本断路器所连**关联元件**（线路名称 / 主变侧别）。
4. **关联保护**：同串/同断路器的线路保护或主变保护是否动作、发跳令时刻（用于交叉核对时序）。

## 二、开关保护动作分析

### 2.1 断路器失灵保护

| 项目 | 内容 |
|------|------|
| 失灵启动条件 | 保护发跳令后本断路器未跳开（电流仍存在） |
| 失灵启动时刻 | 相对线路/主变保护发跳令的时间差（正常 <20~40ms） |
| 联跳矩阵 | 跳哪些相邻断路器（同串另一台断路器 + 相邻元件/对侧） |
| 失灵动作结果 | 联跳成功 / 未动 |

> 失灵联跳须与线路/主变保护发跳令时序交叉核对：发跳令(t) → 失灵启动 → 联跳相邻。时间差过大标 `[待核实: 失灵配合异常]`。

### 2.2 死区保护

| 项目 | 内容 |
|------|------|
| 死区判定 | 故障位于 CT 与断路器之间（差动/距离无法感受） |
| CT 位置 | 线路 CT / 断路器 CT（死区在 CT-断路器区间） |
| 动作结果 | 跳开本断路器及相邻断路器 |

### 2.3 三相不一致保护（非全相）

| 项目 | 内容 |
|------|------|
| 不一致启动 | 单相跳开后长期非全相运行（重合闸把手断开设单相、单相 Ongoing 故障） |
| 延时动作 | 经整定延时后动作跳开 |

### 2.4 充电 / 过流保护（若投）

| 项目 | 内容 |
|------|------|
| 投入场景 | 空充长线、旁路代路时投入的辅助保护（断路器近后备） |
| 动作结果 | 动作 / 未动 |

### 2.5 重合闸分析（若在开关保护，且线路投重合）

| 项目 | 内容 |
|------|------|
| 重合闸位置 | **在断路器保护（开关保护）实现**（3/2 接线特性：一台断路器被两条线路共享，只有开关保护掌握"重合哪条线路、合哪台断路器"） |
| 重合对象 | 哪台断路器 |
| 重合方式 | 单重 / 三重 |
| 重合结果 | 成功 / 失败 / 未动作 / 重合于永久故障→复跳 |

> **注意**：线路保护简报的重合闸章在 500kV+（一般不投）或 3/2 接线下 **N/A**，其重合行为须在本节（开关保护）分析。

## § 开关速查（电压/接线 → 章节启用）

| 章节 | 500kV+（3/2） | 220kV 3/2 接线 | 常规 110kV 及以下 |
|------|------------------|------------------|--------------------|
| 失灵保护（§2.1） | 启用 | 启用 | 一般 N/A |
| 死区保护（§2.2） | 启用 | 启用 | 一般 N/A |
| 三相不一致（§2.3） | 启用 | 启用 | 一般 N/A |
| 充电/过流（§2.4） | 按定值 | 按定值 | 按定值 |
| 重合闸（§2.5） | 在开关保护（若投） | 在开关保护（若投） | 在线路保护 |
| 母差章节 | **N/A**（3/2 无常规母差） | 有母差时启用 | 按接线 |
