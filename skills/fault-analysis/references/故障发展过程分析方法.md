# 故障发展过程分析方法（全故障类型通用）

> **适用范围**：主变 / 线路 / 母差 / 配电设备 **所有故障简报**。
> **强制等级**：简报必须含「故障发展过程分析」专节（见各模板），由 Subagent 在段落中提供「故障发展过程数据」区块，主 Agent 据此解读。
> **目的**：仅凭"故障类型=XX""跳闸元件=YY"不足以判断保护行为是否合理。还原故障**起始→发展→全盛→切除**的真实演化，可识别**发展性故障**（如单相接地→相间→三相）、**电弧性/不稳定短路**、保护动作是否"跟住了"故障演化，以及有无越级/拒动漏判。

---

## 一、数据来源

- **原始采样 CSV**（DAT→CSV 输出，含 `时间ms` 列与三相 V/I 通道），由 Subagent 直接读取计算。
- 提取对象：**故障元件**的三相电压、三相电流（或配电设备的专属量：电容器不平衡电压/差压、电抗器电流、站用变差动电流等）。
- 不同设备取量：
  - 主变：故障侧（高压/中压/低压分支）三相电压 + 三相电流；穿越性故障时可对比各侧。
  - 线路：本端（及双端对端）三相电压 + 三相电流。
  - 母差：故障段母线各间隔电流 + 母线三相电压。
  - 配电设备：设备本体三相电流/电压 + 专属故障量。

---

## 二、分析步骤

### 1. 波形基础确认
- 采样率 `fs = 1000 / 采样间隔ms`（Hz）。
- 是否存在**预故障段**（录波起点电流≈负荷、电压≈额定）：有则取前 50 点作稳态基准；无（DAT 第 0 点已是故障量）则标注"无预故障段，故障在录波起点已存在"，此时"发展过程"以录波内部相对演化还原。

### 2. 关键事件时刻提取（绝对时间，复用 CSV `时间ms`→绝对时刻）
- **故障电流启动**：任一相 `|I|` 首次超过阈值（建议 `>1.5×预故障RMS` 或绝对阈值如 5A 二次）的索引时刻。
- **各相电压塌缩**：各相 `|U|` 首次跌至 `<30%额定` 的时刻（判读接地/故障相别）。
- **发展转折**：电流/电压由"单相特征"转为"多相特征"的时刻（如 B、C 相同时爬升）。
- **切除（电流归零）**：故障相电流回落至 ≈0 的时刻（通常对应跳闸出口后断路器开断）。

### 3. 分阶段 RMS（每段取窗口 RMS，二次值为主、一次值括号附注）
建议划分：
| 阶段 | 判定 | 用途 |
|------|------|------|
| 起始 | 启动后~首 1~2 周波 | 看初始故障相别、是否单相 |
| 发展 | 转折前后 | 看是否扩展、扩展到哪些相 |
| 全盛 | 电流/电压稳定最大段 | 看最终故障类型、幅值 |
| 切除 | 跳闸后~电流归零 | 验证断路器确实开断、有无复燃 |

### 4. 相间关系（全盛期）
- 计算三相电流两两**相关系数** `corr(A,B)`、`corr(B,C)`、`corr(A,C)`：
  - 单相接地/短路：仅故障相大，其余≈负荷，相关性无意义。
  - 两相短路：两故障相电流幅值相近、**反相**（`corr≈-1`），第三相≈负荷。
  - 三相短路：三相均大；理想对称 `corr≈+1/电角度偏移`，实际常因电弧呈非 ±1。
- **电弧性/发展性判据**：不同瞬时呈现 **AB、BC、AC 不同相间对交替主导**，或相关系数介于 -1~+1 之间不收敛 → 经电弧由单相/两相扩展为三相的**不稳定短路**。

### 5. 发展性判读（核心产出）
- **是发展性故障**：起始为单相接地（某相电压→0、电流仅容性/较小），随后 B、C 相电流爬升、最终转为两相/三相短路。须明确"起始类型 → 发展类型"。
- **非发展性（瞬时型）**：录波起点即三相/两相同时短路，无单相先导。
- 对小电流接地系统（35/10kV）：单相接地初始仅容性电流，故障电流小、电压特征明显（故障相电压塌缩、线电压基本对称），**不可误判为相间短路**。
