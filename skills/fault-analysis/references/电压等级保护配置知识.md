# 全电压等级保护配置知识（合并版 · 220kV+ 双套 / 110(66)kV / 35kV / 10(6)kV / 配电设备）

> 本文件是 `电网故障智能分析` 技能的领域知识底座，**覆盖全部电压等级与配置形态**。
> 核心主线有两条：
> 1. **配置套数维度**——220kV+ 为**双套**（每端主后合一 ×2），110kV 及以下为**单套**（每设备 1 套；双端线路每端各 1 套，共 1~2 套）。
> 2. **电压等级维度**——不同电压等级的接线、主/后备保护、重合闸投退、接地方式差异显著，是低电压等级分析的分水岭。
> 主 Agent 与报告模板**必须依据"识别到的电压等级 + 套别 + 接地方式 + 设备类型"启用/跳过相应章节**（见末尾"§9 电压等级开关"）。

## 1. 电压等级差异矩阵（设计基线）

| 电压等级 | 典型接线 | 线路主保护 | 后备保护 | 重合闸 | 接地方式 | 母差 | 配电设备 |
|----------|----------|------------|----------|--------|----------|------|----------|
| **500kV+（含 750kV、1000kV 特高压）** | **3/2 断路器为主**、4/3 | 光纤电流差动（双端纵差，每端 2 套） | 距离(Ⅰ/Ⅱ/Ⅲ) + 零序(Ⅰ/Ⅱ/Ⅲ/Ⅳ) | **一般不投**（系统稳定要求；若投多为单重，且逻辑在**断路器保护/开关保护**中实现） | 大电流接地(直接) | **无常规母差**（3/2 无母线，失灵/死区/联跳替代） | 无 |
| **220kV+** | 3/2 断路器、双母双分段 | 光纤电流差动（双端纵差，每端 2 套） | 距离(Ⅰ/Ⅱ/Ⅲ) + 零序(Ⅰ/Ⅱ/Ⅲ/Ⅳ) | **投**：三相一次/单重/综重 | 大电流接地(直接) | 双母**配母差**（双套） | 一般无 |
| **110kV** | 线变组 / 内桥 / 单母分段 / 双母 | 光纤电流差动（双端纵差，每端 1 套） | 距离(Ⅰ/Ⅱ/Ⅲ) + 零序(Ⅰ/Ⅱ/Ⅲ/Ⅳ) | **投**：三相一次或单重 | 大电流接地(直接) | 单母分段/双母**配母差** | 一般无 |
| **66kV** | 同 110kV 简化 | 光纤差动 / 距离 | 距离 + 零序 / 过流 | 投(架空线) | 大电流接地 | 少配 | 一般无 |
| **35kV** | 单母 / 单母分段 / 线变组 | 重要线光纤差动；一般线距离/过流 | 距离 + 零序过流 | **一般不投**(电缆/辐射) | 小电流接地(消弧线圈/电阻) | 少配 | 电容器/电抗器/站用变 |
| **10(6)kV** | 单母 / 单母分段 | 过流 / 零序过流(**无距离**) | 过流 + 零序过流 | **不投** | 小电流接地(消弧线圈/不接地) | 一般不配(靠进线后备) | **电容器/电抗器/站用变为主**；早期线路常**两相 CT 配置**(A/C 相，B 相无 CT) |

### 1.1 双套（220kV+）vs 单套（110kV 及以下）贯穿全等级的共性

| 维度 | 220kV+（双套） | 110kV 及以下（单套） |
|------|---------------------|---------------------------|
| 配置 | 每端双套（主后合一 ×2） | 每设备**单套** |
| 装置数量 | 线路 = 4 套（2 端 ×2 套） | 线路 = 1~2 套（单端单套 =1；双端单套 =2，每端各 1 套） |
| 一致性校验 | 同侧两套动作/差流一致性 | **无两套概念**；仅双端单套时校验"两端差流一次值相等(<10%)" |
| 测距校验 | 本侧+对侧≈全长 | 双端单套同左；单端单套时单端测距 vs 线路全长(定值/台账)合理性 |
| 时间对齐 | 四套两步对齐 | 双端单套(2 装置)仍两步对齐；单端单套以本装置电流正突变为 t=0 |
| 距离Ⅰ段 | 常退出 | 110(66)kV **常投入(约 80% 范围)** → 不动作判别更关键；35/10kV 无距离 |

> **500kV+ 与 220kV+ 共性/差异**：同属**双套**（每端 2 套），主/后备保护形态相近；但 500kV+ 接线以 **3/2 断路器**为主、**无常规母线**、**重合闸一般不投**且若投其逻辑在**断路器保护（开关保护）**中实现（非线路保护），并需独立分析**断路器失灵/死区/三相不一致/充电**等开关保护动作（见 §6.5）。110kV 及以下为单套，与 220kV+ 套别维度完全不同。
| 接线 | 3/2 断路器、双母双分段 | 线变组、内桥、单母分段、双母线、单母线（低等级） |
| CT/PT | 见 §4 变比库 | 见 §4 变比库 |

## 2. 接线方式要点

- **3/2 断路器（220kV+，500kV+ 为主）**：每条线路连接两台断路器（一个半断路器串），**无常规母线联络概念**。每个断路器配置**独立的断路器保护（开关保护）**，核心功能见 §6.5；线路保护与断路器保护配合——线路保护判出故障并发跳令，由断路器保护执行跳闸、失灵/死区/三相不一致/重合闸（在开关保护）及失灵联跳相邻断路器。
- **线变组**：线路—变压器直接相连，无母线，故**无母差**；线路与主变保护配合，主变中（或低）压侧可作为线路远后备。
- **内桥**：桥断路器连接两变压器，亦无独立母线，**无母差**。
- **单母分段**：一段母线分段，可配**母差**（110kV 常见）。
- **双母线**：两组母线，配**母差**（110kV 重要站 / 220kV+）。
- **单母线（35/10kV）**：一般不配专用母差，靠进线后备保护实现母线故障灵敏度（母线故障由对侧/进线距离/过流切除）。

## 3. CT / PT 变比库（解析校验与一次值换算参考）

| 电压等级 | PT 变比 | 常见 CT 变比 | 说明 |
|----------|---------|--------------|------|
| 220kV+ | 220kV / 100V (或 220/√3 kV) | 1250/5、2500/5、3000/5、2000/5 | 线路/主变高压侧 |
| 110kV | 110kV / 100V (或 110/√3 kV) | 600/5、400/5、800/5、1200/5 | 线路/主变高压侧 |
| 66kV | 66kV / 100V | 400/5、600/5、300/5 | 同 110kV 简化 |
| 35kV | 35kV / 100V | 300/5、200/5、400/5、600/5 | 主变中压侧、35kV 线路 |
| 10(6)kV | 10kV / 100V (或 10/√3 kV) | 600/5、400/5、800/5、1200/5、200/5 | 主变低压侧、10kV 出线、电容器/电抗器/站用变 |

> 一次值换算：一次电流 = 二次电流 × CT 变比；两侧/多套变比不同时，差流一次值须**按各自变比分别换算后再比对**。
> 公式：`一次值(kA) = 二次值(A) × (CT一次额定 / CT二次额定) / 1000`。

## 4. 接地方式判定（分析前置条件）

接地方式决定零序保护逻辑，是低电压等级分析的分水岭。判定来源优先级：
1. **HDR 定值/系统参数**：含"小电流接地""消弧线圈""中性点经电阻""直接接地""大电流接地"等字段；
2. **线路/变压器参数**：中性点接地匝数、消弧线圈补偿度；
3. **故障现象反推**：单相接地后**三相电压仍对称、仅零序电压出现、故障相电流很小** → 小电流接地；**故障相电流显著增大** → 大电流接地。

| 接地方式 | 典型电压等级 | 单相接地故障特征 | 零序保护动作逻辑 |
|----------|--------------|------------------|------------------|
| 大电流接地(直接) | 220/110(66)kV | 故障相电流显著增大，三相电压不对称 | 零序电流保护按段跳闸（Ⅰ/Ⅱ/Ⅲ/Ⅳ段） |
| 小电流接地(消弧线圈/电阻/不接地) | 35/10(6)kV | 故障相电压↓但电流很小，三相电压基本对称，仅出现零序电压 | 多为**选择性接地/选线/报警**，不一定立即跳闸；永久性接地延时候跳，瞬时性接地选线后恢复 |

### 4.1 小电流接地系统关键分析规则（必读）

1. **单相接地故障一般不跳闸**：小电流接地（消弧线圈/电阻/不接地）系统单相接地时故障电流为电容电流（极小，通常仅几安培），按规程保护**不立即跳闸**，仅选线/报警，瞬时性接地选线后恢复、永久性接地延时跳闸。
   - **判相别陷阱**：若录波显示"单相接地"特征（某相电流大、三相电压基本对称仅零序电压出现）却由相过流跳闸，须**复核是否为相间短路误判或发展性/复合故障**，不可直接认定单相接地已正确跳闸。
   - 本技能 108 线路案例：录波 A 相电流大、AB 线电压塌缩，最终判为 **A-B 两相短路（不接地）**——两相短路故障电流大、必须跳闸，与"单相接地不跳"不矛盾；B 相电流=0 系两相 CT 配置所致（见下）。
2. **早期 10kV 线路「两相 CT 配置」**：早期变电站 10kV 出线常仅 A、C 两相装设 CT（B 相不配 CT），录波中 B 相电流通道读数为 0 或派生值，**不代表 B 相无故障或健康**。
   - 判相别须**以电压特征为主**（线电压塌缩定位故障相别）：A-B 两相短路 → AB 线电压塌缩、BC/CA 线电压近正常；A-C 两相短路 → AC 线电压塌缩；单一线电压塌缩即对应两相短路。
   - 电流仅作辅助：两相 CT 配置下 B 相电流恒为 0，不可据此判定"仅 A 相故障"。

> 简报须将"接地方式"写入**故障前置条件**章节；小电流接地下，故障判定须区分"永久性接地(延时跳闸)"与"瞬时性(选线不跳)"，且须区分"单相接地(不跳)"与"相间短路(必跳)"。

## 5. 重合闸投退规则

| 电压等级 | 重合闸 |
|----------|--------|
| 220kV+ | **投**：三相一次重合闸 / 单相重合（环网）/ 综合重合（按定值） |
| 110(66)kV | **投**：三相一次重合闸（辐射/单电源网）或 单相/三相重合（环网/联络线） |
| 35kV | **一般不投**（多为电缆或辐射网，瞬时性故障概率低）；架空线个别投 |
| 10(6)kV | **不投**（电缆网络为主，重合易扩大事故） |

> 模板"重合闸"章节：**仅 110(66)kV 及以上启用**；35/10kV 输出"N/A（该电压等级一般不投重合闸）"。

## 6. 各设备类型要点

### 6.1 线路保护

- **220kV+（双套）**：每端 2 套（主后合一），光纤电流差动（双端纵差）+ 距离(Ⅰ/Ⅱ/Ⅲ) + 零序(Ⅰ/Ⅱ/Ⅲ/Ⅳ) + 重合闸。四套一致性校验（同侧两套、两侧四套差流）。
- **110(66)kV（单套）**：光纤电流差动（主保，双端纵差，每端各 1 套）+ 距离(Ⅰ/Ⅱ/Ⅲ) + 零序(Ⅰ/Ⅱ/Ⅲ/Ⅳ) 后备 + 重合闸。
  - **开关位置信号判定规则（110kV 及以下）**：HWJ（合位继电器）返回 ≠ 开关已跳开——HWJ 回路接于合闸辅助触点，其返回可能仅表示合闸回路中断，非断路器主触头物理分离。**TWJ（跳位继电器）由 0→1 才是开关跳开的确认信号**，因 TWJ 回路接于跳闸位置辅助触点，与主触头机械联动。分析时序时须以 TWJ 变位为开关状态确认依据，HWJ 变位仅作辅助参考。
- **35kV（单套）**：重要线路配光纤差动；一般线路距离/过流 + 零序过流；一般不投重合闸。
- **10(6)kV（单套）**：过流 + 零序过流（**无距离保护**）；不投重合闸。
- **两相 CT 配置（早期 10kV 线路）**：仅 A、C 两相装 CT，B 相无 CT → 录波 B 相电流恒为 0/派生值，**不代表 B 相无故障或健康**；判相别须以**电压特征为主**（A-B 两相短路→AB 线电压塌缩、BC/CA 近正常；A-C 两相短路→AC 线电压塌缩）。
- **双端单套（2 套装置，两端各 1）**：仍做两步时间对齐 + 两端差流一次值比对(<10%)。
- **单端单套（1 套）**：以本装置电流正突变对齐；测距与线路全长(定值/台账)比对合理性。
- **单端单套"装置内双套电流"判定红线（必读，108 线路实测踩坑）**：单端单套装置（仅 1 个录波器）若记录两套电流（如 CFG 中`保护电流1`/`保护电流2`、`电流1`/`电流2`），二者**均为本端本地采样**，**绝非"本侧/对侧"**——
  - 判别硬规则（三者任一满足即判为同端本地量，不可作差流）：① 通道名**无"对侧/远端"字样**；② HDR 定值区**无"差动/纵差/光纤/纵联/导引"任何纵联功能项**（→ 装置不带纵差，本端不可能有对侧电流）；③ 两套 CT 变比**完全相同**（同端双 CT 典型特征，两端通常变比不同）。
  - 二者差值（如 ~0.004A）**仅反映双套采样/变比一致性**，**禁止**当作"差流≈0、呈穿越性、纵差未动正确"的判据——那是**双端单套（两端各 1 装置）**的差流分析，二者不可混淆。
  - 保护电流1/2 的物理含义通常为**保护CT / 测量CT** 或 **线路侧 / 母线侧 CT**（均属本站本端），精确含义对照装置说明书与定值"两相保护电流"项确认，标 `[待核实]`。

### 6.2 主变保护（全电压等级通用，单套）

- 差动保护（主保）+ 高/中/低后备（复压过流、零序过流等），主后合一或分开。
- 非电量保护：瓦斯（本体重/有载）、压力释放、油温、冷却故障等。
- 越级分析：主变各侧后备作为相邻元件远后备；`over_trip_analysis.py` 与 `auto_select_fault_line.py` 支持主变越级跳闸判定（与电压等级无关）。

### 6.3 母差保护（仅 110(66)kV 及以上单母分段/双母；35kV 少配；10kV 不配）

- 大差（除母联以外所有支路和）+ 小差（各段母线所连支路和）。
- 220kV+ 为双套配置；110(66)kV 为单套。线变组/内桥接线无母差 → 模板母差章节 N/A。
- 母差动作须核对：跳闸矩阵、失灵联跳、各支路电流相位/幅值。

### 6.4 配电设备（10/35kV，单套，无重合闸）

- **电容器**：过压/欠压保护、不平衡保护（桥差/开口三角）、差压/差流、过流。
- **电抗器**：过流、零序过流、差动（大容量）。
- **站用变**：差动（若配置）+ 高/低压侧后备过流/零序。
- 接地方式常小电流；故障多为本体或相邻短路，无重合闸。

### 6.5 开关保护（断路器保护 · 500kV+ 3/2 接线关键）

**设备定位**：500kV+（含 750/1000kV）以 3/2 断路器接线为主，每个断路器配置**独立断路器保护（开关保护）**装置，录波常为独立文件（如 `断路器保护`/`开关保护` 录波）。其分析是 500kV+ 与 220kV（重合闸在线路保护）的**最关键区别**。

**核心功能**：
- **断路器失灵保护（failure）**：线路/主变保护发跳令后该断路器未跳开（电流仍存在）→ 失灵启动，**联跳相邻断路器**（同一串另一台断路器 + 相邻元件/对侧），并可启动远方跳闸。
- **死区保护（dead-zone）**：故障点位于 CT 与断路器之间（差动、距离均无法感受）→ 死区保护动作，跳开本断路器及相邻断路器。
- **三相不一致保护（非全相）**：单相跳开后长期非全相运行（如重合闸把手断开设单相、单相 Ongoing 故障）→ 不一致保护经延时动作。
- **充电保护 / 过流保护**：空充长线、旁路代路时投入的辅助保护，作为断路器本体的近后备。
- **重合闸**：**3/2 接线下重合闸逻辑在断路器保护（开关保护）中实现，而非线路保护**——因一个半断路器接线中一台断路器被两条线路共享，只有断路器保护掌握"重合哪条线路、合哪台断路器"的信息。若 500kV 线路配置重合闸（多为单重），其动作/失败应在**开关保护录波**中分析，线路保护简报重合闸章标注 `N/A（重合闸在开关保护）`。

**分析要点（开关保护动作分析章）**：
1. 开关保护录波常为**独立装置文件**，须与线路/主变保护录波**交叉核对时序**：线路/主变保护发跳令（t）→ 断路器保护执行跳闸 / 失灵 / 重合。
2. **失灵联跳**：核查失灵启动条件、联跳矩阵（跳哪些相邻断路器）、失灵动作相对时间；与线路保护发跳令的时间差应合理（<20~40ms 为正常配合）。
3. **重合闸（若投）**：核查重合对象（哪台断路器）、重合方式（单重/三重）、重合于永久故障→复跳。
4. **死区 / 三相不一致**：核查 CT 位置与动作逻辑（死区在 CT-断路器区间；不一致在非全相持续超时）。
5. 与线路保护差流/距离分析解耦：**开关保护不判故障类型**，只负责"执行跳闸 + 失灵/死区/不一致/重合"的断路器侧行为。

> 简报须将"开关保护动作分析"作为 500kV+（及 220kV 3/2 接线）的**独立章节**，与线路/主变保护分析并列；母差章节因 3/2 无常规母线而 **N/A**，改用失灵/死区/三相不一致/联跳分析。

## 7. 可信度原则（合并版 · 双套 + 单套适配）

- **双套（220kV+）保留**：同侧两套差流一致性、两套动作一致性、两侧四套差流一次值基本相等(<10%)。
- **单套适配（110kV 及以下）**：
  - 删除"同侧两套差流一致性""两套动作一致性"两条（不适用单套）。
  - 双端单套(2 装置)：两端差流一次值应基本相等（<10%），偏差标 `[待核实: 差流偏差较大]`。
  - 单端单套(1 装置)：单端测距 vs 线路全长(定值/台账)合理性；无对侧数据不强行"本侧+对侧≈全长"。
  - 距离Ⅰ段不动作判别（仅 110(66)kV 线路）：控制字(距离保护Ⅰ段有效)=1 + 故障点 ≤ Ⅰ段范围 → 应动未动。
- **单端单套"装置内双套电流"红线**：见 §6.1，禁止将装置内两套本地电流差值当作纵差差流。
- **数据来源标注**：HDR 定值/故障信息 vs CSV 计算值，不一致标 `[待核实]`。

## 8. 配置识别前置步骤（必做）

主 Agent 在解析任何录波前，**先**从目录结构、HDR、CFG 识别以下维度，写入简报"故障前置条件"：

1. **电压等级**：PT 变比 / 线路额定电压 / HDR 系统参数 → 220 / 110 / 66 / 35 / 10(6) kV。
   - **判定红线（易错点）**：电压等级须以**被保护元件自身相电压通道**（名称含"相电压"且非"同期""母线"）为准；`同期电压`（同步参考）、`母线电压` 仅为并网/同期或母线参考，**绝不可据此把 10kV 馈线误升为 110kV**（否则会错误启用重合闸/距离Ⅰ段）。典型误判：110kV 变电站内的 10kV 馈线，自身相电压=10kV、同期电压=110kV → 仍判 **10kV 单套**，重合闸 N/A、无距离。
2. **套别（配置数）**：
   - 220kV+ → **双套**（每端 2 套，线路共 4 套）。
   - 110kV 及以下 → **单套**；双端线路每端各 1 套（共 2 套 = 双端单套），单端线路 1 套（= 单端单套）。
   - **硬判据**：单端单套装置若记录两套电流，按 §6.1 红线判定为装置内本地量，非本侧/对侧。
3. **接地方式**：大电流（220/110/66kV）/ 小电流（35/10kV）。
4. **设备类型**：线路 / 主变 / 母差 / 配电设备（电容器/电抗器/站用变）。
5. **接线方式**：3/2、双母、单母分段、线变组、内桥、单母 → 决定是否启用母差章节。
6. **CT 配置**：三相 CT / 两相 CT（早期 10kV，B 相无 CT）。

## 10. 定值语义规则（防止误读）

### 核心原则

**定值名称中的"退出"对象是定值名中的【功能/逻辑】，而非【被保护对象】。** 误将闭锁/监测功能的"退出"理解为保护功能本身的"退出"，是本 Skill 中已确认的高危误读模式。

### 规则

| 定值名称格式 | 含义 | 常见误读 |
|-------------|------|----------|
| `XX闭锁YY = 0-退出` | XX 发生时**不闭锁** YY，YY 保护继续可用 | ❌ "YY 保护被退出了" |
| `XX闭锁YY = 1-投入` | XX 发生时**闭锁** YY，YY 保护被暂锁 | — |
| `XX保护软压板 = 0-退出` | XX 保护被**软压板退出**，确实不可用 | ✓ 这时退出才代表保护不可用 |

### 判定四步法（遇到任何含"退出"的定值）

1. **提取主语**：定值名中"闭锁""闭重""闭差"前面的那个"XX"是什么
2. **提取谓语**：定值名的动词是"闭锁""退出""投入"还是其他
3. **提取宾语**：定值名中"YY"是什么（例如"差动"）
4. **判定**：`XX闭锁YY = 0-退出` → XX 发生时不闭锁 YY → YY 可用。**除非定值名明确是"YY保护软压板=0-退出"或"YY保护控制字=0-退出"，否则"退出"修饰的不是 YY 本身的可用性。**

### 实例

```
CT断线闭锁差动 = 0-退出
  ↓ 主语=CT断线 | 谓语=闭锁 | 宾语=差动
  ↓ CT断线时不闭锁差动
  ↓ 差动保护可用 ✓

CT断线闭锁差动 = 1-投入
  ↓ CT断线时闭锁差动
  ↓ 差动保护在CT断线时被暂锁 ✗

差动保护软压板 = 0-退出
  ↓ 差动保护本身被退出
  ↓ 差动保护不可用 ✓
```

> **绵动线教训**：PSL-621UDA-G `CT断线闭锁差动 = 0-退出` 被误读为"差动保护退出不可用"，导致整份简报将此作为核心结论之一。实际含义为 CT 断线时不闭锁差动，差动保护正常可用（定值 1.5A 已投），其未动作的原因应从故障电流是否满足差动启动条件等方向分析，而非归因于"保护被退出"。
