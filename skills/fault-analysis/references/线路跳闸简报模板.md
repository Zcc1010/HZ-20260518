# 线路跳闸简报模板（全电压等级 · 配置自适应）

> **适用范围**：500kV+（含 750kV、1000kV 特高压，双套·3/2 断路器接线，重合闸一般不投、开关保护独立成章）/ 220kV+（双套，每端 2 套，线路共 4 套）/ 110(66)kV（单套，双端各 1 套或单端 1 套）/ 35kV / 10(6)kV（单套）线路故障录波分析。
> **配置自适应**：本模板一套结构覆盖全部配置形态，按"识别到的电压等级 + 套别 + 接地方式 + CT 配置"启用/跳过相应子章节（见文末 §开关速查）。
> **命名原则**：优先使用 IED 代码（客观）；站名/线路名由用户提供或推断并标 `[待核实]`，不得臆造。
> **判相别优先用电压**：早期 10kV 两相 CT 配置（B 相无 CT，电流恒为 0）时，以**线电压塌缩**定位故障相别，不得仅凭电流判相。
> **⚠ PDF 非默认**：输出仅含 `.md` 简报；非用户明确要求不得自动生成 PDF。
> **⚠ 录波器不入时序表**：故障录波器为监测设备，不得出现在保护动作时序表中；其数据仅用于 §5.4 故障录波评价交叉验证。
> **⚠ 电气量互校默认不列录波器**：电气量表中仅列保护装置；录波器仅在同厂站两套保护电气量不一致（偏差≥10%）时作为仲裁参考引入。
> **⚠ 厂站分开（必遵）**：时序表、一致性分析、评价表均须按厂站分别展开；**同厂站多套分列展示**（列为「相对时刻 / 第1套 / 第2套」），禁止用行标注"装置"列替代分列（详见 §5.2 / §6.1）。
> **⚠ 保护与录波分表**：§二"涉及保护装置"中保护装置与故障录波器为两张独立表，禁止混入同一表中；保护表"厂站名称"列为第二列。录波器表精简列数，标"仅作参考"。

## 报告命名
`YYYY年M月DD日 [线路名称]跳闸简报.md`

## 故障前置条件识别（必做，写在简报开头或 §一）
解析前先判定并写入：
1. **电压等级**：PT 变比 / 线路额定电压 / HDR 系统参数 → **500（含 750/1000kV）** / 220 / 110 / 66 / 35 / 10(6) kV。
2. **套别（配置数）**：
   - 220kV+ → **双套**：厂站1/厂站2 × 第1套/第2套（共 4 套）。
   - 110kV 及以下 → **单套**：双端线路每端各 1 套（共 2 套 = 双端单套），单端线路 1 套（= 单端单套）。
   - **单端单套"装置内双套电流"红线**：单端单套装置若记录两套电流（保护电流1/2），按 `电压等级保护配置知识.md §6.1` 判为装置内本地量，**绝非本侧/对侧**，禁止当作纵差差流。
3. **接地方式**：大电流接地（220/110/66kV）/ 小电流接地（35/10kV，消弧线圈/电阻/不接地）。
4. **接线方式**：3/2、双母、单母分段、线变组、内桥、单母 → 决定是否启用母差章节。
5. **CT 配置**：三相 CT / 两相 CT（早期 10kV，B 相无 CT）。

## 五、动作时序表

### 5.1 时间对齐基准

**双套（220kV+）/ 双端单套（2 装置）：两步对齐法**
- 第一步：电流正突变对齐——各装置故障相电流正突变最大相及其绝对时间；最早者为参考；校正量 = 参考时间 − 该装置正突变时间；无正突变装置用同厂站/同端第一次负突变对齐。
- 第二步：保护启动对齐——校正量应用到各装置保护启动时间；最早校正后保护启动 = t=0 ms；相对时间 = 校正后事件时间 − t=0。

**单端单套（1 装置）**：以本装置电流正突变为 t=0，无需跨装置对齐。

### 5.2 动作时序表

**按厂站分开展示**（必做，双套 / 双端单套均适用）——便于阅读时一眼看出"同一厂站下两套装置动作时序是否同步"，避免把不同厂站的保护挤在一张表里造成误读。

### 6.5 距离Ⅰ段不动作判别  `[电压等级开关：500kV+/220/110(66)kV 线路启用（常退）；35/10kV 无距离 → N/A]`

**触发条件**：装置 Events 中均无"接地距离Ⅰ段动作"信号，或故障判定结论将其列为待核实项。

**判读红线（易错点）**：Ⅰ段投入与否**必须**以**专用分位控制字**为准，不得反推：
- PCS（南瑞）：`距离保护Ⅰ段有效`（=0 退出 / =1 投入）；亦可读 `距离保护Ⅰ段` 控制字。
- WXH/WHX（许继）：`距离保护Ⅰ段`（=0 退出 / =1 投入）。
- **`距离保护投入` / `距离保护软压板` =1 仅为距离元件"总投位"，不代表Ⅰ段分位投入**——220kV 线路距离Ⅰ段**常退**（见 `电压等级保护配置知识.md` §1 开关速查表"距离Ⅰ段=常退出"），总投=1 而Ⅰ段分位=0 属正常既定配置。
- **Ⅰ段定值已整定（如 0.40Ω）亦不代表投入**——"有定值而退出"是常见配置，不动作正确。

| 判别项 | 值 | 结论 |
|--------|----|------|
| 距离保护**总投/软压板** | 距离保护投入={0/1}、距离保护软压板={0/1} | {仅表距离元件总投，不代表Ⅰ段} |
| **Ⅰ段专用控制字**（投入判别位） | {0/1}（PCS `距离保护Ⅰ段有效` / WXH `距离保护Ⅰ段`） | **{退出/已投入}** ← 以此为准 |
| Ⅰ段定值(阻抗) | {X.XX}Ω | — |
| Ⅰ段时间定值 | {X.XX}s | — |
| Ⅰ段有效保护范围(估算) | {X.XX}km | 按 Z1={X.XX} Ω/km 推算 |
| 本端/各套测距 | {各端测距}km | — |
| 故障点 vs Ⅰ段范围 | {在范围外/在范围内} | {描述} |

**结论**：距离Ⅰ段不动作 **{正确 / 待核实: 应动未动}**。
- 控制字=0（退出）→ 无论故障点位置，不动作均正确
- 控制字=1 + 故障点 > Ⅰ段范围 → 正确不动作（物理不可达）
- 控制字=1 + 故障点 ≤ Ⅰ段范围 → 距离Ⅰ段应动未动，必须核实

**距离Ⅰ段动作行为分析（Ⅰ段已动作时）——须按厂站分表** `[双端/双重化案例必用]`：
若装置 Events 中**有**"接地距离Ⅰ段动作"信号（Ⅰ段确实动作），本节改为核对"动作是否正确"，且**必须按厂站分别分析**（与 §5.2 动作时序、§6.1 两套一致性的分表原则一致）。原因：Ⅰ段整定阻抗对应的保护范围两端相同，但**各厂站"本端至故障点"的测距不同**，须逐站判断"本端测距 ≤ 本端Ⅰ段范围"。

- **5.5.1 / 6.5.1 {厂站1}距离Ⅰ段**：列该站各套（第1套/第2套）控制字、Ⅰ段动作时刻、本端测距、本端测距 vs Ⅰ段范围、结论。
- **5.5.2 / 6.5.2 {厂站2}距离Ⅰ段**：同上。
- **汇总表**：厂站 | Ⅰ段控制字 | Ⅰ段动作时刻 | 本端测距 vs 范围 | 结论。

每站结论判据：控制字=1（投入）+ 本端测距 ≤ 本端Ⅰ段范围 + 记录动作 → **正确动作**；控制字=1 + 本端测距 > 范围却动作 → 疑超范围误动，须核实。**不得**把两端测距合并成一张"故障点 31~39km"的笼统表——须逐站给出该站测距与结论。（参考 `analysis/道卧线` 案例 §5.5.1/5.5.2/5.5.3。）

### 6.6 接地方式与零序保护逻辑  `[电压等级开关：按接地方式分支]`

| 接地方式 | 零序保护动作逻辑 |
|----------|------------------|
| 大电流接地（220/110/66kV） | 零序电流保护按段跳闸（Ⅰ/Ⅱ/Ⅲ/Ⅳ段），故障相电流显著增大 |
| 小电流接地（35/10kV） | 多为选择性接地/选线/报警，**未必立即跳闸**；须区分"永久性接地(延时跳闸)"与"瞬时性(选线不跳)" |

> 小电流接地系统下，单相接地后三相电压基本对称、仅零序电压出现、故障相电流很小。故障判定须据此区分永久/瞬时，不可套用大电流接地"零序段跳闸"逻辑；且须区分"单相接地(不跳)"与"相间短路(必跳)"。

### 6.7 开关保护（断路器保护）动作分析  `[电压等级开关：500kV+（3/2 接线）启用；220kV 仅 3/2 接线时启用；常规 110kV 及以下 N/A]`

> **为何独立成章**：500kV+（含 750/1000kV）以 **3/2 断路器**接线为主，每个断路器配置**独立断路器保护（开关保护）**装置（录波常为独立文件）。其与 220kV 的关键区别：**重合闸逻辑在开关保护中实现，而非线路保护**；且须分析失灵/死区/三相不一致/充电等开关侧行为。母差章节因 3/2 无常规母线而 **N/A**，改用本节失灵/死区/三相不一致/联跳分析。

| 分析项 | 内容 / 核查要点 |
|--------|------------------|
| 开关保护录波 | 是否独立文件（如 `断路器保护`/`开关保护` 录波）；与线路保护录波**交叉核对时序**：线路保护发跳令(t) → 断路器保护执行跳闸 / 失灵 / 重合 |
| 断路器失灵保护 | 失灵启动条件、联跳矩阵（跳哪些相邻断路器：同串另一台 + 相邻元件/对侧）、失灵动作相对时间；与线路保护发跳令时间差应合理（<20~40ms 为正常配合） |
| 死区保护 | 故障位于 CT 与断路器之间（差动/距离无法感受）→ 死区动作跳本断路器及相邻；核查 CT 位置与动作逻辑 |
| 三相不一致保护 | 单相跳开后长期非全相运行（重合闸把手断开设单相、单相 Ongoing 故障）→ 不一致经延时动作 |
| 充电/过流保护 | 空充长线、旁路代路时投入的辅助保护（作为断路器近后备） |
| 重合闸（若投） | **在开关保护**：重合对象（哪台断路器）、重合方式（单重/三重）、重合于永久故障→复跳 |

**与线路保护解耦**：开关保护**不判故障类型**，只负责"执行跳闸 + 失灵/死区/不一致/重合"的断路器侧行为；故障类型/差流/距离判据仍在线路保护章分析。

## 一次值换算规则

一次值(kA) = 二次值(A) × (CT一次额定值 / CT二次额定值) / 1000

CT 参数来源：HDR SettingValue 模块。各侧/各间隔 CT 可能不同，分别换算。

## 时间对齐计算方法（两步对齐法）

**第一步：电流正突变对齐** — 各装置故障相电流正突变最大相及其绝对时间；最早者为参考；校正量 = 参考时间 − 该装置正突变时间；无正突变用同厂站/同端第一次负突变对齐。
**第二步：保护启动对齐** — 校正量应用到各装置保护启动时间；最早校正后保护启动 = t=0 ms；相对时间 = 校正后事件时间 − t=0。
