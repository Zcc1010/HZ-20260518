# 继电保护通道信息含义

> 用于grep速查录波文件中的通道信号含义

## 模拟通道

| 通道名关键字 | 说明 |
|-------------|------|
| Ia / IaA / 保护电流A相 | A相保护电流测量 |
| Ib / IbA / 保护电流B相 | B相保护电流测量 |
| Ic / IcA / 保护电流C相 | C相保护电流测量 |
| 3I0 / 保护零序电流 | 零序电流，用于接地故障检测 |
| Ia1 / Ib1 / Ic1 / 正序电流 | 正序分量电流 |
| 3I01 / 零序电流1 | 零序分量电流 |
| IA_R / IB_R / IC_R / 冗余 | 双AD采样自检，同一信号的两路独立采样，用于数据校验和AD故障检测 |
| IDA / IDB / IDC / 差动电流 | 纵差保护用差动电流 |
| Ua / UaV / 保护电压A相 | A相保护电压测量 |
| Ub / UbV / 保护电压B相 | B相保护电压测量 |
| Uc / UcV / 保护电压C相 | C相保护电压测量 |
| 3U0 / 保护零序电压 | 零序电压，用于接地故障检测 |
| UL / 邻线路电压 | 相邻线路电压，用于同期检查、备自投、环网供电等场景 |
| Ux / 线路电压 | 本线路侧电压，用于线路保护、重合闸同期检测 |

## 保护动作流程

```
故障发生 → 保护装置检测
    ↓
【保护启动】保护逻辑判断
    ↓
【纵联差动保护动作】判断故障类型
    ↓
【A/B/C相跳闸动作】确定跳闸相别
    ↓
【A/B/C相跳闸出口】出口继电器动作
    ↓
断路器收到跳闸命令并动作
    ↓
【跳位A/B/C】位置继电器监控
    ↓
【TWJa/b/c_GOOSE】通过网络发送位置状态
```

## 纵联保护信号流向

```
【本侧保护装置】
    │
    ├── 保护逻辑判断 → 跳闸出口 → 断路器
    │                     ↓
    │                 跳位TWJ（位置反馈）
    │
    └── 远传开出 ────[光纤通道]────→ 远传开入（对侧保护）
                                ↓
                          其它保护动作（接收对侧信号）
```
