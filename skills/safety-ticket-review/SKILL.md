# 220kV常规站二次安措票审查

对220kV常规变电站继电保护"二次安全措施票"进行专业审查的完整工作流。核心手段是**硬压板投退 + 端子排解线/封挡**（常规站）。

## 何时使用
- 用户要**生成审查要点**（检查清单）——基于模板或《技术细则》第5章。
- 用户给一张**实际安措票**（.docx），要求**逐条审查并出具审查意见书**。
- 用户要**提取/阅读**安措票 docx 文本。
- 用户发来**新的技术细则/标准**，要求同步进审查要点。

## 资源
- `references/技术细则-第5章.md` — 权威条款原文（5.1检验 / 5.2改造），审查溯源以此为准。**审查前必读**。
- `references/审查要点-常规站.md` — 13章审查要点知识库（票面/票面规范性/原始状态/联跳/采样/信号/交直流/过程/专项速查/14条红线）。**生成与审查时逐条对照**。
- `references/人工审查经验-补充.md` — 用户基于现场经验积累的补充审查依据（站内配置、回路关系等），不受细则条款号约束，作为审查判断参考。
- `references/审查意见书示例.md` — 一份完整审查意见书的示例（主变保护改造），展示审查精度与深度要求：逐行核对端子号、交叉比对电压等级、中性线标错、复制粘贴错误等。**审查时必须达到此示例同等细致程度**。
- `scripts/docx_extract.py` — 无依赖提取 docx 文本（含表格）。
- `scripts/docx_helpers.py` — 中文 DOCX 生成助手（`Doc` 类：title/h/p/check/table 等），保证排版一致。
- `scripts/gen_review_points.py` — 常规站《审查要点》docx 生成器（基于 references 知识库，调参即可复用）。
- `scripts/gen_review_report.py` — 实际安措票《审查意见书》生成器示例（含主变改造审查逻辑，照此改写）。

## 工作流

### 场景A：提取安措票文本
```bash
python3 scripts/docx_extract.py "某某变XX保护更换安全措施票.docx"
```
输出含 `[TABLE]…[/TABLE]` 表格结构，用于理解票面。

### 场景B：生成"审查要点"文档
1. 读 `references/审查要点-常规站.md`（如用户提供新细则/模板，先据此更新知识库要点）。
2. 用 `docx_helpers.Doc` 写生成脚本，按知识库章节输出：封面 → 一~十三（勾选项用 `d.check(text, tag="5.1.1c")`，红线/速查用 `d.table(...)`，结论签字表）。
3. 每条审查项标注对应条款号便于溯源。
4. 生成后用 `docx_extract.py` 抽查 `☐` 数量与章节标题验证完整性。

### 场景C：审查一张实际安措票 → 出具审查意见书（最常用）
1. `docx_extract.py` 提取被审票全文。
2. **判定作业类型**：检验（5.1）还是改造/更换（5.2）；判定设备类型：主变/母差/线路。
3. 对照 `references/审查要点-常规站.md`、`技术细则-第5章.md` 及 `人工审查经验-补充.md` **逐环节核对**：
   票面完整性 → 原始状态 → 联跳 → 电流 → 电压 → 信号 → 交直流 → 过程安措。
4. 每个问题归类为 **必须整改(M)** 或 **建议核实(S)**；标注**高危**（可能误拆/误动运行设备，如端子终端号复制错、中性线标错、漏解运行间隔联跳）。
5. 用 `docx_helpers.Doc` 生成《审查意见书》，结构：
   - 封面 + 元信息表（被审文件/设备作业类型/审查依据/结论）
   - 问题统计（必须整改N｜建议核实M｜高危K）
   - 一、必须整改项（表：编号/位置·条款/问题描述/依据/整改建议）
   - 二、建议核实项（表：编号/位置·条款/问题描述/核实建议）
   - 三、逐章审查明细（■问题 / ✓合格 两类批注，逐章）
   - 四、审查结论 + 分级签字表（编制自审/技术负责人/签发人）
6. 交付：`present_files` 同时给出**审查意见书 + 被审原票**便于对照。

## 审查要点铁律（速记）
- **电流**：先短接、后打开连片，防 CT 二次开路；母差/串接装置在正确侧先短接确认无流；改造须**端子箱+屏后双点**。
- **电压**：打开试验端子连片或解线并封挡，防 PT 二次短路/接地；N600 一并处理；多电压等级（220/110/35kV）勿漏。
- **联跳**：漏退/漏解即误跳运行设备——最高优先级；母差改造不得遗漏任一运行间隔。
- **绝缘**：解线红胶布包好、断空开红胶布封挡、光纤装防尘帽/套。
- **常见高危错误**：电压/电流端子终端号从别的电压等级"复制粘贴"导致标签重复（会误拆运行端子）；中性线 N 误标为相别 A。

## 注意
- 智能站（软压板 SV/GOOSE + 检修压板 + 拔光纤）做法与常规站差异大，本 skill 聚焦常规站；如需智能站版另行区分。
- 具体端子/回路以本站图纸、信息流图、调度要求为最终依据；意见书须提示"执行前对照图纸逐字核对"。
