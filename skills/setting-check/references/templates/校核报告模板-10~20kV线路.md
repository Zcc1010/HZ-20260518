# 10~20kV线路保护定值校核报告模板（专用补充）

> 本文件仅包含 10~20kV 线路保护区别于通用模板的专用内容：设备参数、保护功能清单。
> 校核维度与判断依据见 `校核报告模板-通用.md`，不在此重复。

---

## 设备参数

校核基本信息中设备参数时，10~20kV线路应核对应以下参数：

| 项目 | 说明 |
|------|------|
| 保护CT变比 | 如 600/5 |
| PT变比 | 如 10kV/100V |

> 计算书中应包含CT变比、线路阻抗（或线路长度+单位阻抗）、系统短路容量等参数。核对定值单与计算书中的CT变比是否一致。

---

## 保护功能清单

10~20kV线路保护装置（如PCS-9611A）通常包含以下保护功能，校核概览中按实际投入的功能逐行填写：

| 保护功能 | 关键定值项 | 关键控制字 |
|---------|-----------|-----------|
| 过流Ⅰ段 | 过流Ⅰ段定值（A）、过流Ⅰ段时间（s） | 过流保护Ⅰ段、过流Ⅰ段经方向、过流Ⅰ段经电压（复压闭锁） |
| 过流Ⅱ段 | 过流Ⅱ段定值（A）、过流Ⅱ段时间（s） | 过流保护Ⅱ段、过流Ⅱ段经方向、过流Ⅱ段经电压 |
| 过流Ⅲ段 | 过流Ⅲ段定值（A）、过流Ⅲ段时间（s） | 过流保护Ⅲ段、过流Ⅲ段经方向、过流Ⅲ段经电压 |
| 零序过流Ⅰ段 | 零序过流Ⅰ段定值（A）、零序过流Ⅰ段时间（s） | 零序过流Ⅰ段、零序采用自产零序电流 |
| 零序过流Ⅱ段 | 零序过流Ⅱ段定值（A）、零序过流Ⅱ段时间（s） | 零序过流Ⅱ段 |
| 零序过流告警 | 零序过流告警定值（如有） | 零序过流告警 |
| 过流加速段 | 过流加速段定值（A）、过流加速时间（s） | 过流加速保护、过流加速经电压 |
| 零序过流加速段 | 零序过流加速段定值（A）、零序过流加速时间（s） | 零序过流加速保护 |
| PT断线相过流 | PT断线相过流定值（A）、PT断线相过流时间（s） | PT断线自检 |
| 过负荷告警 | 过负荷定值（A）、过负荷时间（s） | 过负荷告警 |
| 重合闸 | 重合闸时间（s）、同期合闸角（°）、大电流闭重定值（A） | 重合闸方式（检同期/检无压/TWJ启动）、大电流闭锁重合、停用重合闸 |
| 低频减载 | 低频减载频率（Hz）、低频减载时间（s）、低频频率滑差闭锁（Hz/S）、低频电压闭锁（V） | 低频减载、低频减载经滑差闭锁 |
| 低压减载 | 低压减载电压定值（V）、低压减载时间（s）、电压变化率闭锁（V/s） | 低压减载 |
