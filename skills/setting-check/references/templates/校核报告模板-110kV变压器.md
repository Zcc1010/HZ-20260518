# 110kV变压器保护定值校核报告模板（专用补充）

> 本文件仅包含 110kV 变压器保护区别于通用模板的专用内容：设备参数、保护功能清单。
> 校核维度与判断依据见 `校核报告模板-通用.md`，不在此重复。
> 110kV变压器保护结构与220kV变压器类似，差动保护/后备保护配置相近，详见 `校核报告模板-220kV变压器.md`。
> 主要差异：110kV变压器差动速断可靠系数取5（220kV取4），零序过流方向不投入。

---

## 设备参数

### 变压器参数

| 项目 | 说明 |
|------|------|
| 主变额定容量（MVA） | 如 50MVA |
| 高压侧额定电压（kV） | 如 115kV |
| 中压侧额定电压（kV） | 如 38.5kV（三绕组） |
| 低压侧额定电压（kV） | 如 10.5kV |
| 接线方式 | 如 YNd11（双绕组） |
| 中性点接地方式 | 直接接地/经间隙接地/不接地 |

### CT/PT参数

| 项目 | 说明 |
|------|------|
| 高压侧CT变比 | 如 600/5 |
| 高压侧零序CT变比 | 中性点直接接地时使用 |
| 高压侧间隙CT变比 | 中性点经间隙接地时使用 |
| 中压侧CT变比 | 三绕组变压器 |
| 低压侧CT变比 | 如 3000/5 |
| 高压侧PT变比 | 如 110kV/100V |
| 中压侧PT变比 | 如 35kV/100V |
| 低压侧PT变比 | 如 10kV/100V |

> 接带旋转机组的110kV变电站宜保持一台变压器高压侧中性点直接接地；接带逆变器并网电源和无并网电源的110kV变电站，高压侧中性点宜不接地或经间隙接地。

## 保护功能清单

### 主保护

| 保护功能 | 关键定值项 | 关键控制字 |
|---------|-----------|-----------|
| 纵差差动速断 | 纵差差动速断电流定值（Ie），可靠系数宜取5 | 纵差差动速断 |
| 纵差差动保护 | 纵差保护启动电流定值（Ie），宜取0.4Ie | 纵差差动保护、CT断线闭锁差动保护 |
| 二次谐波制动 | 二次谐波制动系数，宜取0.15 | — |
| 比率差动制动 | 比率差动制动系数，宜取0.5 | — |

### 高压侧后备保护

| 保护功能 | 关键定值项 | 关键控制字 |
|---------|-----------|-----------|
| 复压过流Ⅰ段 | 复压过流Ⅰ段定值（A）、1/2时限（s）、低电压闭锁（V）、负序电压闭锁（V） | 复压过流Ⅰ段带方向（方向指向变压器）、经复压闭锁、各时限投退 |
| 复压过流Ⅱ段 | 复压过流Ⅱ段定值（A）、1/2时限（s） | 复压过流Ⅱ段（方向不投）、各时限投退 |
| 零序过流Ⅰ段 | 零序过流Ⅰ段定值（A）、1/2时限（s） | 零序过流Ⅰ段（方向不投）、采用外接零流 |
| 零序过流Ⅱ段 | 零序过流Ⅱ段定值（A）、1/2时限（s） | 零序过流Ⅱ段（方向不投） |
| 间隙过流保护 | 间隙过流时间（s） | 间隙过流（中性点经间隙接地时） |
| 零序过压保护 | 零序过压时间（s） | 零序过压 |

### 中/低压侧后备保护

| 保护功能 | 关键定值项 | 关键控制字 |
|---------|-----------|-----------|
| 复压过流Ⅰ段 | 复压过流Ⅰ段定值（A）、1/2时限（s） | 复压过流Ⅰ段带方向（方向指向本侧母线）、经复压闭锁 |
| 复压过流Ⅱ段 | 复压过流Ⅱ段定值（A）、1/2时限（s） | 复压过流Ⅱ段、各时限投退 |

> 110kV变压器零序过流保护方向不应投入。零序过流Ⅰ段方向指向变压器，中压侧零序过流Ⅰ段方向指向母线（如有）。
