# 220kV变压器保护定值校核报告模板（专用补充）

> 本文件仅包含 220kV 变压器保护区别于通用模板的专用内容：设备参数、保护功能清单。
> 校核维度与判断依据见 `校核报告模板-通用.md`，不在此重复。

---

## 设备参数

校核基本信息中设备参数时，220kV变压器应核对应以下参数：

### 变压器参数

| 项目 | 说明 |
|------|------|
| 主变高中压侧额定容量（MVA） | 如 150MVA |
| 主变低压侧额定容量（MVA） | 如 75MVA（三绕组变压器低压侧容量可能不同于高/中压侧） |
| 中压侧接线方式钟点数 | 如 12（即 Yd12/Yy0），影响差动保护相位补偿 |
| 低压侧接线方式钟点数 | 如 11（即 Yd11），影响差动保护相位补偿 |
| 高压侧额定电压（kV） | 如 230kV |
| 中压侧额定电压（kV） | 如 117kV |
| 低压侧额定电压（kV） | 如 38kV |

### PT参数

| 项目 | 说明 |
|------|------|
| 高压侧PT变比 | 如 220kV/100V |
| 中压侧PT变比 | 如 110kV/100V |
| 低压侧PT变比 | 如 35kV/100V |

### CT参数

| 项目 | 说明 |
|------|------|
| 高压1侧CT变比 | 主变高压侧套管CT或开关CT，如 1200/5 |
| 高压2侧CT变比 | 高压侧备用CT（如未使用标注"未使用"） |
| 高压侧零序CT变比 | 中性点直接接地时使用，如 600/5 |
| 高压侧间隙CT变比 | 中性点经间隙接地时使用，如 400/5 |
| 中压侧CT变比 | 主变中压侧CT，如 1200/5 |
| 中压侧零序CT变比 | 中压侧中性点零序CT |
| 中压侧间隙CT变比 | 中压侧中性点间隙CT |
| 低1分支CT变比 | 低压侧第1分支CT，如 2000/5 |
| 低2分支CT变比 | 低压侧第2分支CT（如未使用标注"未使用"） |
| 公共绕组CT变比 | 自耦变压器公共绕组CT，如 600/5 |
| 公共绕组零序CT变比 | 公共绕组零序CT，如 600/5 |

> 计算书中应包含变压器各侧阻抗参数、系统正序/零序阻抗、接地方式（中性点直接接地/经间隙接地）等。核对定值单与计算书中的容量、CT变比、接线组别是否一致。

## 保护功能清单

220kV变压器保护通常分为主保护和各侧后备保护，校核概览中按实际投入的功能逐行填写：

### 主保护

| 保护功能 | 关键定值项 | 关键控制字 |
|---------|-----------|-----------|
| 纵差差动速断 | 纵差差动速断电流定值（Ie） | 纵差差动速断 |
| 纵差差动保护 | 纵差保护启动电流定值（Ie） | 纵差差动保护、CT断线闭锁差动保护 |
| 工频变化量差动保护（如配置） | — | 纵差工频变化量差动保护（自定义控制字） |
| 零序分量差动保护（如配置） | — | 零序分量差动保护（自定义控制字） |

### 高压侧后备保护

| 保护功能 | 关键定值项 | 关键控制字 |
|---------|-----------|-----------|
| 复压过流Ⅰ段 | 复压过流Ⅰ段定值（A）、1/2/3时限（s）、低电压闭锁（V）、负序电压闭锁（V） | 复压过流Ⅰ段带方向、指向母线、经复压闭锁、各时限投退 |
| 复压过流Ⅱ段 | 复压过流Ⅱ段定值（A）、1/2/3时限（s） | 复压过流Ⅱ段带方向、指向母线、经复压闭锁、各时限投退 |
| 复压过流Ⅲ段 | 复压过流Ⅲ段定值（A）、1/2时限（s） | 复压过流Ⅲ段经复压闭锁、各时限投退 |
| 零序过流Ⅰ段 | 零序过流Ⅰ段定值（A）、1/2/3时限（s） | 零序过流Ⅰ段带方向、指向母线、采用自产零流、各时限投退 |
| 零序过流Ⅱ段 | 零序过流Ⅱ段定值（A）、1/2/3时限（s） | 零序过流Ⅱ段带方向、指向母线、采用自产零流、各时限投退 |
| 零序过流Ⅲ段 | 零序过流Ⅲ段定值（A）、1/2时限（s） | 零序过流Ⅲ段采用自产零流、各时限投退 |
| 间隙过流保护 | 间隙过流时间（s） | 间隙过流 |
| 零序过压保护 | 零序过压时间（s） | 零序过压 |
| 高压侧失灵联跳 | — | 高压侧失灵经主变跳闸 |

### 中压侧后备保护

| 保护功能 | 关键定值项 | 关键控制字 |
|---------|-----------|-----------|
| 复压过流Ⅰ段 | 复压过流Ⅰ段定值（A）、1/2/3时限（s） | 复压过流Ⅰ段带方向、指向母线、经复压闭锁、各时限投退 |
| 复压过流Ⅱ段 | 复压过流Ⅱ段定值（A）、1/2/3时限（s） | 复压过流Ⅱ段带方向、指向母线、经复压闭锁、各时限投退 |
| 复压过流Ⅲ段 | 复压过流Ⅲ段定值（A）、1/2时限（s） | 复压过流Ⅲ段经复压闭锁、各时限投退 |
| 零序过流Ⅰ段 | 零序过流Ⅰ段定值（A）、1/2/3时限（s） | 零序过流Ⅰ段带方向、指向母线、采用自产零流、各时限投退 |
| 零序过流Ⅱ段 | 零序过流Ⅱ段定值（A）、1/2/3时限（s） | 零序过流Ⅱ段带方向、指向母线、采用自产零流、各时限投退 |
| 零序过流Ⅲ段 | 零序过流Ⅲ段定值（A）、1/2时限（s） | 零序过流Ⅲ段采用自产零流、各时限投退 |
| 间隙过流保护 | 间隙过流1/2时限（s） | 间隙过流1/2时限 |
| 零序过压保护 | 零序过压1/2时限（s） | 零序过压1/2时限 |
| 中压侧失灵联跳 | — | 中压侧失灵经主变跳闸 |

### 低压侧后备保护（各分支）

| 公共绕组零序过流 | 公共绕组零序过流定值（A）、时间（s） | 公共绕组零序过流保护跳闸 |
