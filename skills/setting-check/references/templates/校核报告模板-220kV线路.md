# 220kV线路保护定值校核报告模板（专用补充）

> 本文件仅包含 220kV 线路保护区别于通用模板的专用内容：设备参数、保护功能清单。
> 校核维度与判断依据见 `校核报告模板-通用.md`，不在此重复。

---

## 设备参数

校核基本信息中设备参数时，220kV线路应核对应以下参数：

| 项目 | 说明 |
|------|------|
| 保护CT变比 | 如 1250/5 |
| PT变比 | 如 220kV/100V |
| 线路长度（km） | 用于距离保护灵敏度校验（50km以下灵敏度系数≥1.45） |
| 线路阻抗（Ω） | 正序阻抗，用于距离保护整定 |
| 系统短路容量 | 最大/最小方式下的短路电流 |

> 计算书中应包含CT变比、线路阻抗（或线路长度+单位阻抗）、系统短路容量等参数。220kV线路通常配置纵联保护+距离保护+零序方向过流保护。
> 220kV终端线路距离保护灵敏度系数要求：50km以下≥1.45，50~100km≥1.4，100~150km≥1.35，150~200km≥1.3，200km以上≥1.25。

## 保护功能清单

220kV线路保护装置通常包含以下保护功能，校核概览中按实际投入的功能逐行填写：

### 主保护

| 保护功能 | 关键定值项 | 关键控制字 |
|---------|-----------|-----------|
| 光纤电流差动 | 差动电流定值（A）、差动速断定值（A） | 光纤差动保护、差动速断、CT断线闭锁差动 |
| 纵联距离/纵联零序方向 | 纵联零序方向过流定值（A）、纵联距离灵敏角（°） | 纵联距离保护、纵联零序方向保护 |
| 工频变化量距离 | 工频变化量距离定值（Ω） | 工频变化量阻抗元件 |

### 后备保护

| 保护功能 | 关键定值项 | 关键控制字 |
|---------|-----------|-----------|
| 接地距离Ⅰ段 | 接地距离Ⅰ段定值（Ω）、时间（s） | 接地距离Ⅰ段投退 |
| 接地距离Ⅱ段 | 接地距离Ⅱ段定值（Ω）、时间（s） | 接地距离Ⅱ段 |
| 接地距离Ⅲ段 | 接地距离Ⅲ段定值（Ω）、时间（s） | 接地距离Ⅲ段 |
| 相间距离Ⅰ段 | 相间距离Ⅰ段定值（Ω）、时间（s） | 相间距离Ⅰ段投退 |
| 相间距离Ⅱ段 | 相间距离Ⅱ段定值（Ω）、时间（s） | 相间距离Ⅱ段 |
| 相间距离Ⅲ段 | 相间距离Ⅲ段定值（Ω）、时间（s） | 相间距离Ⅲ段 |
| 零序方向过流Ⅰ段 | 零序过流Ⅰ段定值（A）、时间（s） | 零序过流Ⅰ段、零序Ⅰ段带方向 |
| 零序方向过流Ⅱ段 | 零序过流Ⅱ段定值（A）、时间（s） | 零序过流Ⅱ段、零序Ⅱ段带方向 |
| 零序方向过流Ⅲ段 | 零序过流Ⅲ段定值（A）、时间（s） | 零序过流Ⅲ段 |
| 零序方向过流Ⅳ段 | 零序过流Ⅳ段定值（A）、时间（s） | 零序过流Ⅳ段 |
| PT断线过流 | PT断线相过流定值（A）、PT断线相过流时间（s） | PT断线自检 |
| 过负荷告警 | 过负荷定值（A）、过负荷时间（s） | 过负荷告警 |
| 重合闸 | 重合闸时间（s）、同期合闸角（°）、单重长延时（s） | 重合闸方式、单相/三相重合、停用重合 |
| 距离保护振荡闭锁 | 振荡闭锁检测周期（s） | 振荡闭锁（一般不投入） |
