# 35kV变压器保护定值校核报告模板（专用补充）

> 本文件仅包含 35kV 变压器保护区别于通用模板的专用内容：设备参数、保护功能清单。
> 校核维度与判断依据见 `校核报告模板-通用.md`，不在此重复。
> 35kV变压器差动速断可靠系数取7。

---

## 设备参数

| 项目 | 说明 |
|------|------|
| 额定容量（kVA） | 如 20000kVA |
| 高压侧额定电压（kV） | 如 35kV |
| 低压侧额定电压（kV） | 如 10.5kV |
| 接线方式 | 如 Yd11 |
| 阻抗电压（%） | 如 8% |
| 高压侧CT变比 | 如 400/5 |
| 低压侧CT变比 | 如 1500/5 |
| 高压侧PT变比 | 如 35kV/100V |
| 低压侧PT变比 | 如 10kV/100V |

> 35kV变压器高压侧中性点一般不接地或经消弧线圈接地，不配置零序过流保护（低电阻接地系统除外）。

---

## 保护功能清单

### 主保护

| 保护功能 | 关键定值项 | 关键控制字 |
|---------|-----------|-----------|
| 纵差差动速断 | 差动速断电流定值（Ie），可靠系数宜取7 | 差动速断 |
| 纵差差动保护 | 差动启动电流定值（Ie），宜取0.4Ie | 纵差差动保护 |
| 二次谐波制动 | 二次谐波制动系数，宜取0.15 | — |
| 比率差动制动 | 比率差动制动系数，宜取0.5 | — |
| 电流速断保护（未配置差动时） | 速断电流定值（A）、时间（s） | 速断保护 |

### 后备保护

| 保护功能 | 关键定值项 | 关键控制字 |
|---------|-----------|-----------|
| 高压侧复压过流Ⅰ段 | 复压过流Ⅰ段定值（A）、时限（s）、低电压（V）、负序电压（V） | 复压过流Ⅰ段经复压闭锁 |
| 高压侧复压过流Ⅱ段 | 复压过流Ⅱ段定值（A）、时限（s） | 复压过流Ⅱ段 |
| 低压侧过流Ⅰ段 | 过流Ⅰ段定值（A）、时限（s） | 过流保护Ⅰ段 |
| 低压侧过流Ⅱ段 | 过流Ⅱ段定值（A）、时限（s） | 过流保护Ⅱ段 |
| 过负荷告警 | 过负荷定值（A）、过负荷时间（s） | 过负荷告警 |
| 零序过流（低电阻接地系统） | 零序过流定值（A）、时间（s） | 零序过流保护 |
