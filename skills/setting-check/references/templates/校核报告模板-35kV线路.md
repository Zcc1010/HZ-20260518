# 35kV线路保护定值校核报告模板（专用补充）

> 本文件仅包含 35kV 线路保护区别于通用模板的专用内容：设备参数、保护功能清单。
> 校核维度与判断依据见 `校核报告模板-通用.md`，不在此重复。
> 35kV线路保护功能与10~20kV线路类似，参见 `校核报告模板-10~20kV线路.md`。

---

## 设备参数

校核基本信息中设备参数时，35kV线路应核对应以下参数：

| 项目 | 说明 |
|------|------|
| 保护CT变比 | 如 600/5 |
| PT变比 | 如 35kV/100V |
| 线路长度（km） | |
| 线路阻抗（Ω） | 正序阻抗 |

> 35kV线路为小电流接地系统（中性点不接地或经消弧线圈接地），一般不配置零序方向保护，以过流保护为主。
> 部分低电阻接地系统的35kV线路配置零序过流保护，按接地变零序过流保护配合整定。

---

## 保护功能清单

35kV线路保护装置通常包含以下保护功能，校核概览中按实际投入的功能逐行填写：

| 保护功能 | 关键定值项 | 关键控制字 |
|---------|-----------|-----------|
| 过流Ⅰ段（速断） | 过流Ⅰ段定值（A）、过流Ⅰ段时间（s） | 过流保护Ⅰ段、过流Ⅰ段经方向、过流Ⅰ段经电压（复压闭锁） |
| 过流Ⅱ段 | 过流Ⅱ段定值（A）、过流Ⅱ段时间（s） | 过流保护Ⅱ段、过流Ⅱ段经方向、过流Ⅱ段经电压 |
| 过流Ⅲ段 | 过流Ⅲ段定值（A）、过流Ⅲ段时间（s） | 过流保护Ⅲ段、过流Ⅲ段经方向、过流Ⅲ段经电压 |
| 零序过流（低电阻接地系统） | 零序过流定值（A）、零序过流时间（s） | 零序过流保护 |
| 过流加速段 | 过流加速段定值（A）、过流加速时间（s） | 过流加速保护、过流加速经电压 |
| PT断线相过流 | PT断线相过流定值（A）、PT断线相过流时间（s） | PT断线自检 |
| 过负荷告警 | 过负荷定值（A）、过负荷时间（s） | 过负荷告警 |
| 重合闸 | 重合闸时间（s） | 重合闸方式 |
| 低频减载 | 低频减载频率（Hz）、低频减载时间（s） | 低频减载 |
