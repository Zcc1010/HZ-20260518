# 500kV变压器保护定值校核报告模板（专用补充）

> 本文件仅包含 500kV 变压器保护区别于通用模板的专用内容：设备参数、保护功能清单。
> 校核维度与判断依据见 `校核报告模板-通用.md`，不在此重复。
> 500kV变压器保护结构与220kV变压器类似，详见 `校核报告模板-220kV变压器.md`。
> 主要差异：差动速断可靠系数取5；低压侧无总开关时采用分相差动保护（非纵联差动）；零序差动保护不投入。

---

## 设备参数

### 变压器参数

| 项目 | 说明 |
|------|------|
| 主变高/中压侧额定容量（MVA） | 如 750MVA（自耦变压器） |
| 主变低压侧额定容量（MVA） | 如 240MVA |
| 高压侧额定电压（kV） | 如 525kV |
| 中压侧额定电压（kV） | 如 230kV |
| 低压侧额定电压（kV） | 如 66kV |
| 接线方式 | 自耦变压器 YNa0d11 |

### CT/PT参数

| 项目 | 说明 |
|------|------|
| 高压侧CT变比 | 如 3000/1 |
| 高压侧零序CT变比 | 如 1000/1 |
| 中压侧CT变比 | 如 3000/5 |
| 中压侧零序CT变比 | — |
| 低压侧CT变比 | 如 4000/5（各分支） |
| 公共绕组CT变比 | 自耦变压器专用 |
| 公共绕组零序CT变比 | — |
| 高/中/低压侧PT变比 | 如 500kV/100V, 220kV/100V, 66kV/100V |

---

## 保护功能清单

### 主保护

| 保护功能 | 关键定值项 | 关键控制字 |
|---------|-----------|-----------|
| 分相差动速断 | 差动速断电流定值（Ie），可靠系数宜取5 | 差动速断 |
| 分相差动保护 | 差动启动电流定值（Ie），宜取0.4Ie | 分相差动保护、CT断线闭锁差动 |
| 分侧差动保护 | 分侧差动启动电流定值（Ie），宜取0.4Ie | 分侧差动保护 |
| 二次谐波制动 | 二次谐波制动系数，宜取0.15 | — |
| 比率差动制动 | 比率差动制动系数，宜取0.5 | — |
| 零序差动保护 | **不投入** | 零序差动保护（应退出） |

### 各侧后备保护

> 500kV变压器各侧后备保护（复压过流、零序过流、间隙过流等）与220kV变压器配置类似，详见 `校核报告模板-220kV变压器.md`。
