# 接地变保护定值校核报告模板（专用补充）

> 本文件仅包含接地变保护区别于通用模板的专用内容。
> 校核维度与判断依据见 `校核报告模板-通用.md`，不在此重复。

---

## 设备参数

| 项目 | 说明 |
|------|------|
| 接地变容量（kVA） | 如 500kVA |
| 额定电压（kV） | 如 35kV / 10kV |
| 接地方式 | 中性点经小电阻接地/经消弧线圈接地 |
| 接地电阻（Ω） | 低电阻接地系统，如 12Ω |
| 额定电流（A） | 接地变高压侧额定电流 |
| CT变比 | 如 100/5 |
| 零序CT变比 | 中性点零序CT |

> 低电阻接地系统经接地变中性点接地时，保护动作宜联跳供电变压器低压侧断路器。
> 接地变保护装置应具备"相电流消零"功能。

---

## 保护功能清单

| 保护功能 | 关键定值项 | 关键控制字 |
|---------|-----------|-----------|
| 速断过流（过流Ⅰ段） | 速断过流定值（A）、时间（s，宜取0s），按躲低压侧故障整定 | 速断保护 |
| 过流Ⅰ段 | 过流Ⅰ段定值（A）、时间（s），按躲最大负荷电流整定 | 过流保护 |
| 高压侧零序过流Ⅰ段 | 零序过流Ⅰ段定值（A）、时间（s） | 零序过流Ⅰ段（跳接地变高压侧断路器） |
| 高压侧零序过流Ⅱ段 | 零序过流Ⅱ段定值（A）、时间（s），宜联跳供电变压器低压侧断路器 | 零序过流Ⅱ段 |
| 过负荷告警 | 过负荷定值（A）、过负荷时间（s） | 过负荷告警 |
| 非电量保护（温度/瓦斯等） | 由接地变本体实现 | — |

> 速断过流：按躲接地变低压侧故障整定（Kk≥1.3），按躲励磁涌流整定（Kk=10），时间取0s。优先满足躲低压侧故障原则。
> 过流Ⅰ段：按躲接地变最大负荷电流整定（Kk≥1.5），按低压侧两相故障有灵敏度校验（Ksen≥1.3）。
