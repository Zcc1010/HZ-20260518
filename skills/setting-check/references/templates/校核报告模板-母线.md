# 母线保护定值校核报告模板（专用补充）

> 本文件适用于 110kV、220kV、35kV及以下母线保护。
> 校核维度与判断依据见 `校核报告模板-通用.md`，不在此重复。

---

## 设备参数

| 项目 | 说明 |
|------|------|
| 母线电压等级 | 如 220kV / 110kV / 35kV |
| 母线接线方式 | 双母线/双母线单分段/单母线分段/3/2接线 |
| 基准CT变比 | 母差保护用统一CT变比 |
| 母联CT变比 | 母联（分段）CT变比 |
| 各支路CT变比 | 母线上各元件（线路/变压器）CT变比 |

> 母线保护中的电流定值二次值均基于基准CT变比折算。

---

## 保护功能清单

### 母线差动保护

| 保护功能 | 关键定值项 | 关键控制字 |
|---------|-----------|-----------|
| 差动启动电流 | 差动启动电流定值（A），按小方式母线故障有灵敏度（Ksen≥2）整定 | 差动保护 |
| 比率差动制动 | 比率制动系数斜率 | 比率制动 |
| CT断线告警 | CT断线告警定值（A），宜取CT额定电流的2%~10% | CT断线告警 |
| CT断线闭锁 | CT断线闭锁定值（A），宜取CT额定电流的5%~10% | CT断线闭锁差动 |
| 充电过流保护 | 充电过流定值（A）、充电过流时间（s） | 充电过流保护 |

### 母联（分段）保护

| 保护功能 | 关键定值项 | 关键控制字 |
|---------|-----------|-----------|
| 母联充电过流Ⅰ段 | 充电过流Ⅰ段定值（A）、时间（s，宜取0s） | 充电过流Ⅰ段（正常运行宜退出） |
| 母联充电过流Ⅱ段 | 充电过流Ⅱ段定值（A）、时间（s，宜取0.2s） | 充电过流Ⅱ段 |
| 母联充电零序过流 | 充电零序过流定值（A）、时间（s，宜取0.2s） | 充电零序过流 |
| 母联失灵保护 | — | 母联失灵保护 |
| 母联过流（独立） | 过流定值（A）、时间（s） | 母联独立过流（正常运行宜退出） |

> 母联（分段）独立过流保护正常运行时宜退出。投入时应与变压器后备保护及出线保护配合整定。
