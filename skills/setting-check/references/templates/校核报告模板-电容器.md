# 电容器保护定值校核报告模板（专用补充）

> 本文件仅包含电容器保护区别于通用模板的专用内容。
> 校核维度与判断依据见 `校核报告模板-通用.md`，不在此重复。

---

## 设备参数

| 项目 | 说明 |
|------|------|
| 电容器容量（kvar） | 如 20000kvar |
| 额定电压（kV） | 如 10kV / 35kV |
| 额定电流（A） | 电容器额定电流 |
| 串联电抗器（%） | 如 5% / 6% |
| CT变比 | 如 1500/5 |
| 零序CT变比 | 低电阻接地系统 |

---

## 保护功能清单

| 保护功能 | 关键定值项 | 关键控制字 |
|---------|-----------|-----------|
| 过流Ⅰ段 | 过流Ⅰ段定值（A）、时间（s，宜取0.2s），按躲冲击电流整定（Kk=3~5） | 过流保护Ⅰ段 |
| 过流Ⅱ段 | 过流Ⅱ段定值（A）、时间（s，宜取0.5s），按躲额定电流整定（Kk=2） | 过流保护Ⅱ段 |
| 零序过流Ⅰ段（低电阻接地系统） | 零序过流Ⅰ段定值（A）、时间（s，宜取0.3s） | 零序过流Ⅰ段 |
| 零序过流Ⅱ段（低电阻接地系统） | 零序过流Ⅱ段定值（A）、时间（s，宜取0.6s），与接地变零序过流Ⅱ段配合 | 零序过流Ⅱ段 |
| 过压保护 | 过压定值（V）、过压时间（s） | 过压保护 |
| 欠压保护 | 欠压定值（V）、欠压时间（s） | 欠压保护 |
| 不平衡电压保护 | 不平衡电压定值（V）、时间（s） | 不平衡电压保护 |
| 不平衡电流保护 | 不平衡电流定值（A）、时间（s） | 不平衡电流保护 |

> 过流Ⅰ段：按躲电容器冲击电流整定（Kk=3~5），500kV变电站电容器还应与上级变压器低绕组过流保护配合（Kk≥1.2，分支系数0.67）。
