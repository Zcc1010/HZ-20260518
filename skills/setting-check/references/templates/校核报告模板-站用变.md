# 站用变保护定值校核报告模板（专用补充）

> 本文件仅包含站用变保护区别于通用模板的专用内容。
> 校核维度与判断依据见 `校核报告模板-通用.md`，不在此重复。
> 站用变保护整定原则同接地变保护。

---

## 设备参数

| 项目 | 说明 |
|------|------|
| 站用变容量（kVA） | 如 315kVA |
| 高压侧额定电压（kV） | 如 35kV / 10kV |
| 低压侧额定电压（kV） | 如 0.4kV |
| 接线方式 | 如 Dyn11 |
| CT变比 | 如 100/5 |

> 站用变低压侧（0.4kV）一般为三相四线制，不纳入定值校核范围，仅校核高压侧保护。

---

## 保护功能清单

| 保护功能 | 关键定值项 | 关键控制字 |
|---------|-----------|-----------|
| 电流速断 | 速断电流定值（A）、时间（s，宜取0s），整定原则同接地变 | 速断保护 |
| 过流Ⅰ段 | 过流Ⅰ段定值（A）、时间（s，宜取0.5s），按躲最大负荷电流整定 | 过流保护Ⅰ段 |
| 零序过流Ⅰ段（低电阻接地系统） | 零序过流Ⅰ段定值（A）、时间（s，宜取0.3s），按灵敏度整定（Ksen≥2） | 零序过流Ⅰ段 |
| 零序过流Ⅱ段（低电阻接地系统） | 零序过流Ⅱ段定值（A）、时间（s，宜取0.6s），与接地变零序过流Ⅱ段配合 | 零序过流Ⅱ段 |
| 过负荷告警 | 过负荷定值（A）、过负荷时间（s），整定原则同接地变 | 过负荷告警 |
| 非电量保护（温度/瓦斯等） | 由站用变本体实现 | — |

> 速断保护：整定原则同接地变速断过流保护（躲低压侧故障，Kk≥1.3；躲励磁涌流，Kk=10）。
> 零序过流Ⅱ段：与接地变零序过流Ⅱ段配合，配合系数不小于1.1。
